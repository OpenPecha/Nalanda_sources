\chapter{༄༅། །བདུད་རྩི་འཁྱིལ་བའི་སྒྲུབ་ཐབས།}༄༅༅། །རྒྱ་གར་སྐད་དུ། ཨ་མྲྀ་ཏ་ཀུཎྜ་ལི་སཱ་དྷ་ན།\footnote{ནཾ།  སྣར་ཐང་།  པེ་ཅིན། } བོད་སྐད་དུ། བདུད་རྩི་འཁྱིལ་པའི་སྒྲུབ་\footnote{བསྒྲུབ་  སྣར་ཐང་།  པེ་ཅིན། }ཐབས། བགེགས་ཀྱི་དགྲ་ལ་ཕྱག་འཚལ་ལོ། །བདུད་རྩི་འཁྱིལ་པའི་རྣལ་འབྱོར་དང་བཟླས་པས་བར་དུ་གཅོད་པའི་བགེགས་ཞི་བར་འདོད་པ་དེས། དང་པོ་སྙིང་རྗེ་ཆེན་པོ་བསྒོམ་པར་བྱའོ། །དེ་ནས་བརྟན་པ་དང་གཡོ་བ་ཐམས་ཅད་སྟོང་པ་ཉིད་དུ་བསྒོམ་པར་བྱའོ། །དེ་ནས་རང་གི་སེམས་ཉིད་ཉི་མའི་དཀྱིལ་འཁོར་བསྒོམ་པར་བྱའོ། །དེའི་སྟེང་དུ་ཡི་གེ་ཧཱུཾ་སྔོན་པོ་བསྒོམ་པར་བྱའོ། །དེ་ལས་ཁྲོ་བོའི་རྡོ་རྗེ་ཧཱུཾ་གིས་མཚན་པ་བསྒོམ་པར་བྱའོ། །དེ་ལས་འོད་ཟེར་འཕྲོས་ཏེ་གདུག་པ་ཐམས་ཅད་ཞི་བའི་ཕྲིན་ལས་མཛད་ནས་སླར་དེ་ཉིད་དུ་བསྡུའོ། །དེ་ནས་དེ་རྣམས་གྱུར་པ་ལས། །བགེགས་ཀྱི་དགྲ་ནི་སྔགས་པས་བསྒོམ། །ཨུཏྤལ་སྔོན་པོའི་འདབ་ལྟར་སྔོ། །ཞལ་གསུམ་པ་ལ་ཕྱག་དྲུག་པ། །ཞལ་ནི་སྔོ་དང་དཀར་དང་དམར། །ཞལ་གདངས་པ་ནི་རྣམ་པར་འགྱུར། །གསུས་ཁྱིམ་ཆེ་ཞིང་སྤྱན་གསུམ་པ། །འབར་བའི་འོད་ཀྱིས་རྣམ་པར་བརྒྱན། །སྣ་ཚོགས་རྡོ་རྗེ་འཁོར་ལོ་དང་། །གཏུན་ཤིང་རྣམས་ནི་གཡས་པ་ན། །སྡིགས་མཛུབ་ཞགས་པར་བཅས་པ་དང་། །དྲིལ་བུ་དགྲ་སྟ་གཡོན་ན་བསྣམས། །གཡོན་:བརྐྱང་པ་ཡི་\footnote{བརྐྱང་པ་ཡིས་  སྣར་ཐང་།  པེ་ཅིན། }གནས་པ་ཡིས། །པདྨ་ཉི་མ་ལ་གནས་པའོ། །དེ་ནས་སྙིང་གར་ཨ་\footnote{ཨཱ་  ཅོ་ནེ། }ལས་པདྨ་འདབ་མ་བརྒྱད་པ་བསྒོམ་པར་བྱ། དེའི་སྟེང་དུ་ཧཱུཾ་ལས་ཉི་མའི་དཀྱིལ་འཁོར་བསྒོམ་པར་བྱའོ། །དེའི་སྟེང་དུ་ཧཱུཾ་ལས་རང་དང་འདྲ་བའི་ཡེ་ཤེས་སེམས་དཔའ་ཆ་:ལྔ་ལ་\footnote{ལྔ་པ་  སྣར་ཐང་།  པེ་ཅིན། }བསྒོམ་པར་བྱའོ། །དེའི་སྟེང་དུ་ཉི་མའི་དཀྱིལ་འཁོར་གྱི་སྟེང་དུ་ཏིང་ངེ་འཛིན་སེམས་དཔའ་ཡི་གེ་ཧཱུཾ་བསྒོམ་པར་བྱའོ། །དེ་ལས་འོད་ཟེར་འཕྲོས་ཏེ་རང་བཞིན་གྱི་གནས་ནས་ཁྲོ་བོའི་རྒྱལ་པོ་སྤྱན་དྲངས་ལ་མཆོད་པ་དང་བསྟོད་པ་བྱའོ། །སྤྱན་བགྲང་མཆེ་གཙིགས་སྨིན་མ་བསྡུས། །ཐུགས་ནི་མི་རྟོག་པ་ལ་བཞུགས། །མགུར་གྱི་ཕྱོགས་ནི་ཧཱུཾ་སྒྲས་འགེངས། །ཁྲོ་རྒྱལ་བདུད་རྩི་འཁྱིལ་ལ་བསྟོད། །ཅེས་བྱ་བའོ། །:དེ་ལ་\footnote{དེ་  སྣར་ཐང་།  པེ་ཅིན། }ཛཿ་ཧཱུཾ་བཾ་ཧོཿ་བཞིས། །དགུག་པ་དང་ནི་གཞུག་པ་དང་། བཅིང་པ་དང་ནི་མཉེས་པར་བྱ། །དེ་ནས་གནས་གསུམ་དུ་ཡི་གེ་གསུམ་གྱིས་བྱིན་གྱིས་བརླབ་པར་བྱའོ། །དེ་ནས་རགས་པ་དང་ཕྲ་བའི་རྣལ་འབྱོར་བསྒོམ་པར་བྱའོ། །\footnote{། །རྣལ་འབྱོར་གསལ་ནས་བཟླས་པ་བྱའོ། །  སྣར་ཐང་།  པེ་ཅིན། }དེ་ལ་དང་པོ་ཡི་གེ་ཉི་ཤུ་དྲུག་པའི་སྔགས་ཀྱིས་བསྲུང་བར་བྱའོ། །དེའི་སྔགས་ནི། ཨོཾ་ཧུ་ལུ་ཧུ་ལུ། ཏིཥྛ་:ཏིཥྛ། བྷནྡྷ་བྷནྡྷ།\footnote{ཏིཥྛ།བནྡྷ་བནྡྷ།  སྣར་ཐང་།  པེ་ཅིན། } ཧ་ན་ཧ་ན། ད་ཧ་ད་ཧ། ཨ་མྲྀ་ཏེ་ཧཱུཾ་ཕཊ། འབྱུང་བོ་:ཆེ་གེ་མོ་མཱ་ར་ཡ།\footnote{མོ་ལས་རཀྵ་རཀྵ་ཧཱུཾ།  སྣར་ཐང་།  པེ་ཅིན། } དེའི་འཁོར་ལོ་ནི་གྲོ་ག་ལ་སོགས་པ་ལ་སྣ་ཚོགས་རྡོ་རྗེ་ལྟེ་བར་\footnote{བ་  སྣར་ཐང་།  པེ་ཅིན། }རྒྱ་ཆེར་བྲི་བར་བྱའོ། །འཁོར་ལོའི་དབུས་སུ་ཡི་གེ་ཧཾ་བྲི་བར་བྱའོ། །དའི་ནང་དུ་བསྲུང་བར་བྱ་བའི་ཡི་གེ་བྲི་བར་བྱའོ། །དེ་ནས་འཁོར་ལོའི་རྩིབས་དང་པོ་ལ་ཨོཾ། གཉིས་པ་ལ་ཧུ་ལུ་ཧུ་ལུ། གསུམ་པ་ལ་ཏིཥྛ་ཏིཥྛ། བཞི་པ་:ལ་བྷནྡྷ་བྷནྡྷ། \footnote{ལ་བནྡྷ་བནྡྷ་  སྣར་ཐང་།  པེ་ཅིན། }ལྔ་པ་ལ་ཧ་ན་ཧ་ན། དྲུག་པ་ལ་ད་ཧ་ད་ཧ། བདུན་པ་ལ་ཨ་མྲྀ་ཏ། རྩིབས་བརྒྱད་པ་ལ་ཧཱུཾ་ཕཊ་ཅེས་བྱ་བ་རྣམས་བྲི་བར་བྱའོ། །འདི་ནི་སྔགས་ཀྱི་ཡི་གེ་དགོད་པའོ། །དེ་ནས་རང་གི་མདུན་དུ་ཟ་མ་ཏོག་ལ་སོགས་པ་\footnote{པར་  སྣར་ཐང་།  པེ་ཅིན། }ལེགས་པར་བཞག་ལ་མཆོད་པ་བྱའོ། །དེ་ནས་སྔ་མ་བཞིན་དུ་རང་ཉིད་ལྷའི་སྐུ་བསྐྱེད་དེ། སྔགས་སྔ་མ་ཉིད་འབུམ་ཕྲག་གཅིག་བཟླས་པར་བྱའོ། །དེ་ནས་འཁོར་ལོ་དེ་ཉིད་ལུས་ལ་བཏགས་ན། ལུས་ངག་ཡིད་གསུམ་གྱི་ཕྱི་དང་ནང་གི་འཚེ་བ་ཐམས་ཅད་ལས་བསྲུང་བར་འགྱུར་རོ། །དེ་ལྟར་བདག་ཉིད་སྔོན་དུ་:སྲུང་བ་\footnote{བསྲུང་བ་  སྣར་ཐང་།  པེ་ཅིན། }བྱས་ལ་དེ་ནས་གནོད་པར་:བྱ་བའི་\footnote{བྱེད་པའི་  སྣར་ཐང་།  པེ་ཅིན། }འབྱུང་པོ་བསྐྲད་པར་བྱའོ། །དེའི་སྔགས་ནི་འདི་ཡིན་ནོ། །:ན་མཿ་ས་\footnote{ན་མ་ས་  སྣར་ཐང་།  པེ་ཅིན། }མནྟ་ཀཱ་:ཡ་བཱཀ་ཙིཏྟ་བཛྲཱ་ཎཱཾ།\footnote{ཡ་བཀྩིཏྟ་བཛྲ་ཎཾ  སྣར་ཐང་། ཡ་བཀྩིཏྟ་བཛྲ་ནཾ།  པེ་ཅིན། } ཨོཾ་ཨ་མྲྀ་ཏ་ཀུཎྜ་ལི། ཁ་ཁ་ཁཱ་ཧི་ཁཱ་ཧི། ཏིཥྛ་ཏིཥྛ། བནྡྷ་བནྡྷ། ཧ་ན་ཧ་ན། ད་ཧ་ད་ཧ། གརྫ་གརྫ། བི་སྥོ་ཊ་ཡ་བི་སྥོ་ཊ་ཡ། སརྦ་བིགྷྣཱ། པི་ན་ཡ་ཀཾ། མ་ཧཱ་བཛྲ་ཀྲོ་དྷ།\footnote{ཏ།  སྣར་ཐང་།  པེ་ཅིན། } མ་ཧཱ་:དོཾ་ཥྚྲོདྐ་\footnote{དཾཥྚོད་ཀ་  སྣར་ཐང་།  པེ་ཅིན། }ཊ་བྷཻ་ར་བ་ཡ། ཨ་སི་མུ་ས་ལ་བ་ར་ཤུ་པཱ་ཤ་ཧསྟ་\footnote{ཧ་ཏ་  སྣར་ཐང་།  པེ་ཅིན། }ཡ། མ་ཧཱ་ག་ཎ་པ་ཏི་ཛྲཱི་བི་:ཏཱ་ནནྟ། ཀཱ་\footnote{ཏ་ཨནྟ་ཀ་  སྣར་ཐང་།  པེ་ཅིན། }ར་ཡ་:ཧཱུཾ་ཕཊ་\footnote{སྭཱ་ཧཱ་  སྣར་ཐང་།  པེ་ཅིན། }སྭཱ་ཧཱའི་སྔོན་དུ་འགྲོ་བས་བདག་ལ་གནོད་པར་བྱེད་པའི་འབྱུང་པོ་ཆེ་གེ་མོ་:ཨུཙྪ་ཏ་\footnote{ཨུ་ཙ་ཊ་  སྣར་ཐང་།  པེ་ཅིན། }ཡ་\footnote{ཡ་ཧཱུཾ་  སྣར་ཐང་།  པེ་ཅིན། }ཞེས་བཟླས་ཤིང་ཉུངས་ཀར་གྱི་རྒྱལ་པོ་ཡི་གེ་ཧཱུཾ་དུ་བསམས་ཏེ་དེ་ལས་བྱུང་བའི་ཕུར་བུ་སྟོད་བདུད་རྩི་འཁྱིལ་པའི་དབྱིབས་ཅན་ལྟེ་བ་མན་ཆད་ཕུར་བུ་རྩེ་གཅིག་པར་བསམས་ལ་བརྡེག་པར་བྱའོ། །
བསྙེན་པའི་དུས་སུ་འབུམ་ཕྲག་གསུམ་བཟླས་པར་བྱའོ། །བསྐྲད་པའི་:ལམ་གྱི་\footnote{ལས་ཀྱི་  སྣར་ཐང་།  པེ་ཅིན། }དུས་སུ་སྐེ་ཚེ་དང་། ལན་ཚྭ་དང་། དུག་དང་། :ད་དུ་\footnote{དྷ་དྷུ་  སྣར་ཐང་།  པེ་ཅིན། }རའི་ཁུ་བ་དང་། མཛུབ་མོའི་ཁྲག་རྣམས་བསྲེས་ཏེ་དུར་ཁྲོད་ཀྱི་རས་ལ་རླུང་གི་དཀྱིལ་འཁོར་ཟླ་གམ་ངོས་ལ་རྡོ་རྗེས་མཚན་པ་བྲིས་ལ། དེའི་དབུས་སུ་སྔགས་འབྱུང་པོའི་མིང་དང་སྤེལ་བ་བྲིས་ནས་རྣམ་པར་ཤེས་པ་དགུག་པ་སྔོན་དུ་འགྲོ་བས་མནན་ལ་སྔ་མ་བཞིན་དུ་བཟླས་ནས་དུར་ཁྲོད་དུ་:སྦས་ན་\footnote{སྦས་ནས་  སྣར་ཐང་། }ངེས་པར་འཆི་བར་འགྱུར་རོ། །དེ་ལྟར་བསྐྲད་ཀྱང་མི་འགྲོ་ན་དེ་ལ་མངོན་སྤྱོད་བྱའོ། །དེ་ལ་དང་པོར་བསྙེན་པའི་དུས་སུ་སྔགས་འབུམ་དུ་བཟླས་པར་བྱའོ། །ན་མཿ་\footnote{མཿ་མ་  སྣར་ཐང་།  པེ་ཅིན། }ས་མནྟ་ཀཱ་ཡ་བཱ་ཀ་ཙིཏྟ་བཛྲཱ་\footnote{བཛྲ་  སྣར་ཐང་།  པེ་ཅིན། }ཎཱཾ། ཨོཾ་ཧུ་ལུ་ཧུ་ལུ། ཏིཥྛ་ཏིཥྛ། :བྷནྡྷ་བྷནྡྷ།\footnote{བན་དྷ་བན་དྷ།  སྣར་ཐང་།  པེ་ཅིན། } ཧ་ན་ཧ་ན། ད་ཧ་ད་ཧ། གརྫ་གརྫ། བི་སྥོ་ཊ་ཡ་བི་སྥོ་ཊ་ཡ། སརྦ་བིགྷྣཱཾ།\footnote{བིགྷྣན།  སྣར་ཐང་།  པེ་ཅིན། } བི་ན་ཡ་ཀན། མ་ཧཱ་ག་ཎ་བ་ཏི་ཋཱི་བི་ཏ་ཨནྟ་:ཀཱ་ར་\footnote{ཀ་ར་  སྣར་ཐང་།  པེ་ཅིན། }ཡ་ཧཱུཾ་ཕཊ། བདག་ལ་གནོད་པར་བྱེད་པའི་འབྱུང་པོ་ཆེ་གེ་མོ་མཱ་ར་ཡ་ཧཱུཾ་ཕཊ། ཅེས་པ་བཟླས་སོ། །དེ་ནས་ལས་སྦྱོར་གྱི་དུས་སུ་རོ་བསྲེགས་ཐལ་བ་:ཡུངས་ཀར་\footnote{ཡུངས་མར་  སྣར་ཐང་།  པེ་ཅིན། }དང་བསྲེས་པ་ལས་འབྱུང་པོའི་གཟུགས་བྱས་ལ། :\footnote{དེའི་  སྣར་ཐང་།  པེ་ཅིན། }རྣམ་པར་ཤེས་པ་ནང་དུ་བཅུག་ལ། ཁྲོ་བོའི་སྔགས་སྟོང་རྩ་བརྒྱད་བཟླས་ན་:འཆིང་བར་\footnote{འཆི་བར་  སྣར་ཐང་།  པེ་ཅིན། }འགྱུར་རོ། །གཟུགས་བརྙན་ལ་རྣམ་པར་ཤེས་པ་དགུག་པའི་མན་ངག་ནི། འབྱུང་པོའི་རྣམ་པར་ཤེས་པའི་སྙིང་ག་ན་ཉི་མའི་དཀྱིལ་འཁོར་གྱི་སྟེང་ན། ཡི་གེ་ཧཱུཾ་གི་གཟུགས་གནས་པ་ལ་:ལྟེ་བར་\footnote{ལྟེ་བ་  སྣར་ཐང་།  པེ་ཅིན། }རླུང་གི་:དཀྱིལ་འཁོར་གྱིས་\footnote{དཀྱིལ་འཁོར་  སྣར་ཐང་།  པེ་ཅིན། }བཏེག་པར་བསམ་མོ། །དེ་རང་གི་ཏིང་ངེ་འཛིན་སེམས་དཔའ་ལས་འོད་སྤྲོས་ཏེ། གཟུགས་བརྙན་གྱི་སྙིང་གར་ཞུགས་པར་བསམ་མོ། །དེ་ལྟར་གནོད་བྱེད་འབྱུང་པོ་ལ། །བདག་ཉིད་:སྲུང་བར་\footnote{བསྲུང་བར་  སྣར་ཐང་།  པེ་ཅིན། }བྱས་ནས་ནི། །བསྐྲད་དང་མངོན་སྤྱོད་ལས་བྱ་བ། །འདུས་པའི་རྒྱུད་ཀྱི་མན་ངག་ནི། །སློབ་དཔོན་ཀླུ་སྒྲུབ་ལས་རྙེད་པ། །ཟླ་བ་གྲགས་པ་ཞེས་བྱས་བཤད། །བདུད་རྩི་འཁྱིལ་པའི་རྣལ་འབྱོར་དང་བགེགས་བསལ་བའི་ཆོ་ག་རྫོགས་སོ།། །།སློབ་དཔོན་ཤྲདྡྷཱ་\footnote{ཤྲདྡྷ་  སྣར་ཐང་།  པེ་ཅིན། }ཀ་ར་ཝ་རྨ་དང་། ལོ་ཙཱ་བ་རིན་ཆེན་བཟང་པོས་བསྒྱུར་བའོ།། །།
