\chapter{༄༅། །དཔལ་ནག་པོ་ཆེན་པོའི་བསྟོད་པ་རྐང་པ་བརྒྱད་པ།}༄༅༅། །རྒྱ་གར་སྐད་དུ། ཤྲཱི་མ་ཧཱ་ཀཱ་:ལ་སྱ་ཥྚ་\footnote{ལ་སྱཱཥྚ་  སྣར་ཐང་།  པེ་ཅིན། }མནྟྲ་སྟོ་ཏྲ་ནཱ་མ། བོད་སྐད་དུ། དཔལ་ནག་པོ་ཆེན་པོ་ལ་བསྟོད་པ་རྐང་པ་བརྒྱད་པ་ཞེས་བྱ་བ། དཔལ་ནག་པོ་ཆེན་པོ་ལ་ཕྱག་འཚལ་ལོ། །ཨོཾ་གྲུབ་པར་གྱུར་ཅིག །ཧཱུཾ་ཧཱུཾ་ཕཊ་ཅེས་དྲག་པོའི་སྒྲ་ཡིས་སྲིད་པ་གསུམ་གྱི་ཁོང་ནི་མ་ལུས་འགེངས་ནུས་པའི། །:ཧ་ཧ་ཊ་ཊ་ཞེས་བཞད་སྒྲ་ཡིས་\footnote{ཧཱ་ཧཱ་ཊྚཱ་ཊྚ་ཞེས་བཞད་  སྣར་ཐང་། ཧཱ་ཧཱ་ཊྚ་ཊྚ་ཞེས་བཞད་  པེ་ཅིན། }གང་ཞིག་དུས་ཀུན་དུ་ནི་ཤིན་ཏུ་འཇིགས་:མཛད་ཅིང་།\footnote{མཛད་པ།  སྣར་ཐང་།  པེ་ཅིན། } །ཀཾ་ཀཾ་ཀཾ་ཞེས་ཐོད་པའི་ཕྲེང་བས་དབུ་ལ་སྤྲས་ཤིང་ནག་པོའི་མཐུ་\footnote{མཆུ་  སྣར་ཐང་།  པེ་ཅིན། }དང་མཚུངས་པའི་སྐུ། །བྷྲུཾ་བྷྲུཾ་བྷྲུཾ་ཞེས་ཁྲོས་པའི་མཆོག་གི་ཁྲོ་གཉེར་འཇིགས་མཛད་འཇིགས་:པའི་ཁྭ་\footnote{པའི་  སྣར་ཐང་།  པེ་ཅིན། }གདངས་ཤ་ཟ་ཞིང་། །དབུ་སྐྲ་དང་ནི་སྨ་\footnote{རྨ་  སྣར་ཐང་།  པེ་ཅིན། }ར་ཆེས་\footnote{ཆེར་  སྣར་ཐང་།  པེ་ཅིན། }སེར་ཉེ་བར་སྤྱོད་པའི་ཞིང་སྐྱོང་ཁྱོད་ཀྱིས་སྲུང་བར་མཛོད། །རཾ་རཾ་རཾ་ཞེས་:སྤྱན་མར་\footnote{སྤྱན་དམར་  སྣར་ཐང་།  པེ་ཅིན། }འཁྲུག་ཅིང་བསྒྱུར་\footnote{སྒྱུར་  སྣར་ཐང་།  པེ་ཅིན། }མཛད་ཀྲུཾ་ཀྲུཾ་ཀྲུཾ་ཞེས་རབ་སྒྲོགས་སྤྱན་གྱིས་གཟིགས། །སྨིན་མ་སེར་ཞིང་མཆེ་གཙིགས་རོ་ཡི་གདན་ལ་དགྱེས་པའི་ཞིང་སྐྱོང་ཁྱོད་ཀྱིས་ཞིང་སྐྱོང་མཛོད། །ཧ་ཧ་ཧཱུཾ་དང་ཀཱི་\footnote{ཀི་  སྣར་ཐང་།  པེ་ཅིན། }ལི་ཀཱི་\footnote{ཀི་  སྣར་ཐང་།  པེ་ཅིན། }ལི་ཞེས་སྒྲོགས་ཕྱག་གཡོན་ཁ་ཊྭཱཾ་གར་བཅས་ཐོད་པ་བསྣམས། །རུ་རུ་རུ་ཞེས་ཁྲག་རྒྱུན་འབབ་པ་ཕྱག་གིས་བསྣམས་ཤིང་གསོལ་:བ་འཐུངས་པ་\footnote{དེ་འཐུང་བ་  སྣར་ཐང་།  པེ་ཅིན། }ལ་དགྱེས་ཤིང་། །ཁཾ་ཁཾ་ཁཾ་ཞེས་གཏུམ་པའི་ཕྱག་གཡས་གྲི་གུག་རལ་གྲི་བགེགས་རྣམས་ལ་ནི་རོལ་:མཛད་པའི།\footnote{མཛད་པ།  སྣར་ཐང་།  པེ་ཅིན། } །:ཊཾ་ཊཾ་ཊཾ་\footnote{ཌཾ་ཌཾ་ཌཾ་  སྣར་ཐང་།  པེ་ཅིན། }ཞེས་ཌ་མ་རུ་ཅན་འདི་ཡིས་འདུལ་ཚད་\footnote{མཛད་  སྣར་ཐང་།  པེ་ཅིན། }འཁོར་བཅས་ཞིང་སྐྱོང་ཁྱོད་ཀྱིས་སྲུངས། །རབ་ཏུ་རྔམ་ཞིང་མགོ་བོ་རྣམས་ཀྱི་ཕྲེང་བས་གཤིན་རྗེ་དང་མཚུངས་འཇིགས་པའི་སྐུ་བརྒྱན་ཅིང་། །ཀྵཾ་ཀྵཾ་ཀྵཾ་\footnote{ཀྵཱིཾ་ཀྵཱིཾ་ཀྵཱིཾ་  སྣར་ཐང་།  པེ་ཅིན། }ཞེས་བཟོད་པའི་:ཐུགས་ཅན་\footnote{གཟུགས་ཅན་  སྣར་ཐང་།  པེ་ཅིན། }ཅ་ཅོ་སྒྲོགས་པར་བྱེད་པའི་གདུག་པ་རྣམས་བཟུང་\footnote{གཟུང་  སྣར་ཐང་།  པེ་ཅིན། }སྟེ། །གཅིག་པུས་:སྣ་ཚོགས་མནན་ཅིང་མནན་ནས་ཧ་\footnote{སྣ་ཚོགས་བསྣན་ཅིང་བསྣན་ནས་  སྣར་ཐང་། སྣ་ཚོགས་བསྣན་ཅིང་བསྣན་ནས་ཀ་ཧ་  པེ་ཅིན། }ཀ་:ཧ་ཀ་\footnote{ཧ་ཀ་ཧ་  སྣར་ཐང་། ཧ་  པེ་ཅིན། }བརྗོད་མཛད་ཆར་སྤྲིན་སྔོན་པོའི་མདོག །སྐུ་ལ་མི་ཡི་ཤ་དང་:རྒྱུ་མས་བྲེལ་\footnote{རྒྱུ་མ་སྦྲེལ་  སྣར་ཐང་།  པེ་ཅིན། }བའི་:རྒྱན་གྱིས་\footnote{ཕྲེང་བས་  སྣར་ཐང་།  པེ་ཅིན། }ཀུན་ནས་བརྒྱན་ཅིང་ཞལ་དུ་གསོལ། །དྲག་ཤུལ་ཕྱག་གིས་འདོད་པའི་གཟུགས་དང་མི་སྡུག་གཟུགས་ཅན་གྱི་ནི་འབྱུང་པོ་རོ་ལངས་ཚོགས།\footnote{ཆགས།  སྣར་ཐང་།  པེ་ཅིན། } །མ་ལུས་བཟུང་\footnote{གཟུང་  སྣར་ཐང་།  པེ་ཅིན། }ནས་མྱུར་དུ་བསད་པ་རྣམས་ཀྱི་:མགོ་བོའི་\footnote{མགོ་བོ་  སྣར་ཐང་།  པེ་ཅིན། }ཁྲག་རྒྱུན་འབབ་པ་རབ་ཏུ་གསོལ། །ཧྲིཾ་:ཀྵིཾ་ཤྲཱིཾ་\footnote{ཀྵུཾ་ཤྲིཾ་  སྣར་ཐང་།  པེ་ཅིན། }གི་སྔགས་ཀྱི་གསུངས་ཅན་པ་ཙ་པ་ཙའི་སྔགས་ཀྱིས་བགེགས་རྣམས་རབ་ཏུ་སྲེག །རོལ་པས་རོལ་པ་སེལ་ཞིང་མི་ཡིས་གང་བའི་མཚོ་\footnote{ས་  སྣར་ཐང་།  པེ་ཅིན། }ལ་ཞིང་སྐྱོང་ཁྱོད་ཀྱིས་:སྲུང་བར་\footnote{བསྲུང་བར་  སྣར་ཐང་།  པེ་ཅིན། }མཛད།\footnote{མཛོད།  སྣར་ཐང་།  པེ་ཅིན། } །ཕེཾ་ཕེཾ་ཕཊ་ཅེས་སྒྲོགས་པ་\footnote{པས་  སྣར་ཐང་།  པེ་ཅིན། }སོ་སོར་བསྐྱོད་པའི་མེ་དཔུང་ཆེན་པོའི་དབུས་སུ་བཞུགས་\footnote{གཞུགས་  པེ་ཅིན། }ནས་སུ། །རིགས་ཀྱི་ལུས་ཅན་:སྲུང་བར་\footnote{སྐྱོད་པར་  སྣར་ཐང་།  པེ་ཅིན། }མཛོད་ཅིག་སྒྲུབ་པོ་:རྣམས་ཀྱི་\footnote{རྣམས་ཀྱིས་  སྣར་ཐང་།  པེ་ཅིན། }ཞིང་རྣམས་ངེས་པར་སྐྱོང་:མཛོད་ཅིག །\footnote{མཛོད། །  སྣར་ཐང་།  པེ་ཅིན། }ཕྱུགས་རྣམས་ཀྱིས་ནི་ཉིན་རེ་བཞིན་དུ་བགེགས་དང་སྡིག་འཇོམས་དྲི་མེད་:བརྙེས་པ་\footnote{མཉེས་པ་  སྣར་ཐང་།  པེ་ཅིན། }ཆུ་ཉི་བཞིན། །ཙཾ་ཙཾ་ཙཾ་ཞེས་གཏུམ་པོའི་ཤུགས་ཀྱིས་རབ་གཡོས་རྨད་བྱུང་འོད་ཀྱིས་འཇིག་རྟེན་སྣང་མཛད་པ། །:ཀྲིཾ་ཀྲིཾ་ཀྲིཾ་\footnote{ཀྲཾ་ཀྲཾ་ཀྲཾ་  སྣར་ཐང་།  པེ་ཅིན། }ཞེས་གཤེགས་པས་དགྲ་བོ་ཉོན་མོངས་པ་རྣམས་ངེས་པར་ཉོན་མོངས་འཇོམས་མཛད་ཅིང་། །སཾ་སཾ་སཾ་ཞེས་ཚོགས་པའི་བདག་ཉིད་:དག་ཐོས་ཅིང་\footnote{དམ་ཚིག་ཐོགས་  སྣར་ཐང་།  པེ་ཅིན། }འཛིན་ཞིང་སྐྱོང་གང་ཡིན་ཁྱོད་ཀྱིས་སྐྱོངས། །བཾ་བཾ་བཾ་ཞེས་གཤིན་རྗེ་ལྟར་འགྲོ་\footnote{ཁྲོ་  སྣར་ཐང་།  པེ་ཅིན། }དྲི་མ་མེད་པའི་སྣ་ཚོགས་ཆུ་བཞིན་གཟིགས་མཛད་ཅིང་། །ཡཾ་ཡཾ་ཡཾ་ཞེས་རླུང་གི་ཤུགས་ཀྱིས་མྱུར་དུ་རྒྱུ་ཞིང་ཉོན་མོངས་འཇིག་རྟེན་གནོད་བྱེད་མཁྱེན།
