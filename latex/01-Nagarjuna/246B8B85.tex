\chapter{༄༅། །རྟོག་པ་ལས་གསུངས་པའི་ལྷ་མོ་རིགས་བྱེད་མའི་སྒྲུབ་པའི་ཐབས།}༄༅༅། །རྒྱ་གར་སྐད་དུ། :ཀལ་པོཀྟམ་\footnote{ཀལ་བ་ཊ་ཀུ་རུ་  སྣར་ཐང་། ཀལྤ་ཌ་ཀུ་མ་  པེ་ཅིན། }ཀུ་རུ་ཀུལླ་\footnote{ལེ་  སྣར་ཐང་། ཀུལླེ་  པེ་ཅིན། }སཱ་དྷ་ནཾ། བོད་སྐད་དུ། :རྟོག་པ་\footnote{རྟོགས་པ་  སྣར་ཐང་། }ལས་གསུངས་པའི་ལྷ་མོ་རིགས་\footnote{རིག་  སྣར་ཐང་། }:བྱེད་མའི་\footnote{བྱེད་  པེ་ཅིན། }སྒྲུབ་པའི་ཐབས། འཕགས་མ་རིགས་\footnote{རིག་  སྣར་ཐང་།  པེ་ཅིན། }བྱེད་མ་ལ་ཕྱག་འཚལ་ལོ། །གང་ཞིག་དྲན་པ་ཙམ་གྱིས་ནི། །ལྷ་དང་ལྷ་མིན་མིར་བཅས་པ། །ངེས་པར་:ཅི་བགྱི་\footnote{ཅི་གྱིས་  སྣར་ཐང་། }བྱེད་པ་ཡི། །སྒྲོལ་མ་དེ་ལ་ཕྱག་:འཚལ་ནས།\footnote{འཚལ་ལོ།  སྣར་ཐང་། } །:དང་པོ་\footnote{དང་པོར་  སྣར་ཐང་།  པེ་ཅིན། }རེ་ཞིག་སྔགས་པས་ཡིད་དང་རྗེས་སུ་མཐུན་\footnote{འཐུན་  སྣར་ཐང་། }པའི་གནས་སུ། བུད་མེད་ཀྱི་སྐྱེ་བོ་དང་ཚོགས་པ་ལ་སོགས་པས་དབེན་པར་ནང་པར་སྟན་བདེ་བ་ལ་འདུག་སྟེ། རང་གི་སྙིང་གར་དབྱངས་ཡིག་དང་པོ་ཡོངས་སུ་གྱུར་པའི་ཟླ་བའི་དཀྱིལ་འཁོར་གྱི་སྟེང་དུ་འོད་ཟེར་དང་བཅས་པའི་ཡི་གེ་ཧྲཱིཿ་ས་བོན་ལས་བྱུང་བའི་འོད་ཟེར་དམར་པོའི་ཚོགས་ཀྱིས་འགྲོ་བ་གསལ་བར་བྱས་པའི་:སྔོན་དུ་\footnote{མངོན་དུ་  སྣར་ཐང་། }འགྲོ་བའི་:རིགས་བྱེད་\footnote{རིག་བྱེད་  སྣར་ཐང་། }མའི་:རི་ལ་\footnote{རིག་པ་  སྣར་ཐང་། }གནས་པའི་:རིགས་བྱེད་\footnote{རིག་བྱེད་  སྣར་ཐང་། }མ། བཅོམ་ལྡན་འདས་མ་བསྐུལ་ནས་སྤྱན་དྲངས་ནས་མདུན་གྱི་ནམ་མཁའ་ལ་ཡང་དག་པར་བལྟས་ཏེ། སྙིང་གའི་:ས་བོན་\footnote{ས་བོན་གྱིས་འོད་  སྣར་ཐང་། }ལས་སྤྲུལ་པའི་མེ་ཏོག་དང་སྤོས་ལ་སོགས་པའི་མཆོད་པ་སྣ་ཚོགས་ཀྱིས་མཆོད་པར་བྱའོ། །དེའི་རྗེས་ལ་ཕྱག་:བཙལ་བ་\footnote{འཚལ་བ་  པེ་ཅིན། }དང་། སྡིག་པ་བཤགས་པ་དང་། བསོད་ནམས་ལ་རྗེས་སུ་ཡི་རང་བ་དང་། བསྔོ་བ་དང་། གསུམ་ལ་སྐྱབས་སུ་འགྲོ་བ་དང་། བྱང་ཆུབ་ཏུ་སེམས་བསྐྱེད་པ་སྔོན་དུ་འགྲོ་བས། དཀོན་མཆོག་གསུམ་ལ་སྐྱབས་སུ་མཆི། །སྡིག་པ་ཐམས་ཅད་:སོ་སོར་\footnote{སོའི་སོ་  སྣར་ཐང་། }བཤགས། །འགྲོ་བའི་བསོད་ནམས་རྗེས་ཡི་རང་། །སངས་རྒྱས་བྱང་ཆུབ་ཡིད་:ཀྱིས་གཟུང་།\footnote{ཀྱི་བཟུང་།  སྣར་ཐང་། } །ཞེས་\footnote{། །  སྣར་ཐང་།  པེ་ཅིན། }ལན་:གསུམ་གྱི་\footnote{གསུམ་གྱིས་  སྣར་ཐང་། }བར་དུ་བརྗོད་དོ། །དེ་ནས་ཚངས་པའི་གནས་བཞི་བསྒོམ་པ་སྔོན་དུ་འགྲོ་བས་ཆོས་ཐམས་ཅད་:སྟོང་པ་ཉིད་\footnote{སྟོང་པ་  སྣར་ཐང་། }རྣམ་:པར་བསྒོམ་\footnote{པར་ཉིད་བསྒོམས་  སྣར་ཐང་། }པར་བྱའོ། །:དེ་ནས་\footnote{དེ་  སྣར་ཐང་།  པེ་ཅིན། }བྱིན་གྱིས་\footnote{གྱི་  སྣར་ཐང་། }བརླབ་པའི་སྔགས་བརྗོད་པར་བྱའོ། །ཨོཾ་ཤཱུ་ནྱ་ཏཱ་ཛྙཱ་ན་བཛྲ་སྭ་བྷཱ་:ཝ་ཨཱཏྨ་ཀོ྅\footnote{ཝ་ཨད་མ་ཀོ་  སྣར་ཐང་། ཝ་བ་ཨཱཏྨ་ཀོ་  པེ་ཅིན། }ཧཾ་ཞེས་སོ། །དེ་ནས་འགྲོ་བ་སྡུག་བསྔལ་:མཐའ་དག་\footnote{མཐའ་དག་ལ་  སྣར་ཐང་། }སྙིང་རྗེ་བསྒོམས་ལ། རང་གི་སེམས་:ས་བོན་གྱི་\footnote{ས་བོན་གྱིས་  སྣར་ཐང་། }གཟུགས་སུ་བལྟས་ཏེ། དེ་ཡོངས་སུ་གྱུར་པ་ལས་ཨུཏྤ་ལ་དམར་པོའི་སྟེང་དུ་:དབྱངས་ཡིག་དང་པོ་\footnote{དབྱངས་ཡིག་  སྣར་ཐང་།  པེ་ཅིན། }ཡོངས་སུ་གྱུར་པའི་ཟླ་བ་ལ་གནས་པའི་:ས་བོན་གྱི་\footnote{ས་བོན་གྱིས་  སྣར་ཐང་། }འོད་ཟེར་རྣམས་ཀྱིས་སྔོན་བཞིན་དུ་འགྲོ་བ་སྣང་བར་བྱ་སྟེ། དེ་བཞིན་གཤེགས་པ་བྱེ་བ་དཔག་ཏུ་མེད་པ་རྣམས་བསྐུལ་ཏེ། སྤྱན་:དྲངས་ལ་\footnote{དྲངས་  སྣར་ཐང་། }བསྟིམ་པར་བྱ་སྟེ། དེ་ལ་ཨུཏྤ་\footnote{ཨུཏྤ་ཨུད་པ་  སྣར་ཐང་། }ལ་ལ་སོགས་པ་ཡོངས་སུ་གྱུར་པ་ལས་སྐུ་མདོག་དམར་པོ་པདྨ་དམར་པོ་དང་ཟླ་བའི་དཀྱིལ་འཁོར་ལ་བཞུགས་\footnote{ཞུགས་  སྣར་ཐང་། }པ། ན་བཟའ་དམར་པོས་སྟོད་གཡོགས་བྱས་པ། འོད་ཟེར་དམར་པོ་དང་ལྡན་པ། རིན་པོ་ཆེའི་:རྣ་རྒྱན་\footnote{བརྒྱན་པ་  སྣར་ཐང་། }དང་ལྡན་པ། དབུ་རྒྱན་\footnote{བརྒྱན་  སྣར་ཐང་། }དང་ལྡན་པ། ཕྱག་བཞི་པ། གཡས་པས་མི་འཇིགས་པ་སྦྱིན་པ། གཉིས་པ་\footnote{པས་  སྣར་ཐང་།  པེ་ཅིན། }མདའ་འགེངས་པ། གཡོན་པས་རིན་པོ་ཆེའི་གཞུ་འཛིན་པ། གཉིས་པས་ཨུཏྤ་ལ་\footnote{ཨུད་པ་  སྣར་ཐང་། }དམར་པོ་འཛིན་པ། འོད་དཔག་\footnote{དཔག་ཏུ་  སྣར་ཐང་། }མེད་པའི་དབུ་རྒྱན་ཅན། :རིགས་བྱེད་\footnote{རིག་བྱེད་  སྣར་ཐང་། }མའི་རི་ཕུག་ན་གནས་པའི་སྒྲ་གཅན་གྱི་:མགོ་བོ་\footnote{མགོ་  སྣར་ཐང་། }ལ་གནས་པའི་ཆུང་མ་དང་བཅས་པའི་འདོད་པའི་ལྷའི་སྟེང་ན་གནས་པ། སྒེག་པ་ལ་སོགས་པའི་ཉམས་:འཕེལ་བའི་དབང་\footnote{འཕེལ་བ་  སྣར་ཐང་།  པེ་ཅིན། }དང་ལྡན་པ། དང་པོའི་ལང་ཚོ་དང་ལྡན་པའི་རིགས་བྱེད་མར་བསྒོམ་པར་བྱའོ། །དེ་ལྟར་དམ་ཚིག་\footnote{ཚིགས་  སྣར་ཐང་། }སེམས་དཔའ་བསྐྱེད་ནས་སླར་ཡང་སྐད་ཅིག་:ཙམ་གྱིས་\footnote{ཙམ་གྱི་  སྣར་ཐང་། }ཡེ་ཤེས་སེམས་དཔའ་མདུན་དུ་བཀུག་:ལ་ཡང་དག་པར་\footnote{ལ་  སྣར་ཐང་།  པེ་ཅིན། }མཆོད་དེ། :དམ་ཚིག་གི་\footnote{དམ་ཚིག་  སྣར་ཐང་། }ཕྱག་རྒྱས་ཞལ་ནས་གཞུག་\footnote{གཞུགས་  སྣར་ཐང་། }པར་བྱའོ། །དེ་ལ་དམ་ཚིག་གི་ཕྱག་རྒྱ་ནི་འདི་ཡིན་ཏེ། ཐལ་མོ་སྦྱར་ལ་གུང་མོ་གཉིས་རྩེ་མོར་བྱས་ལ་སྲིན་ལག་གཉིས་ནང་དུ་གཞུག །མཛུབ་\footnote{འཛུབ་  སྣར་ཐང་། }མོ་གཉིས་གུང་མོའི་རྒྱབ་ཏུ་སྦྱར་ལ། :མཐེབ་ཆེན་\footnote{མཐེབ་མོ་  སྣར་ཐང་།  པེ་ཅིན། }གཉིས་\footnote{གཉིས་ཕྱེད་  སྣར་ཐང་།  པེ་ཅིན། }འོག་ཏུ་བྱས་པ་\footnote{ལ་  སྣར་ཐང་། }དེ་ལྟ་བུའི་རྣམ་པའི་ཨུཏྤ་\footnote{ཨུད་པ་  སྣར་ཐང་། }ལའི་ཕྱག་རྒྱའོ། །དེའི་རྗེས་སུ། བྱང་ཆུབ་:རྡོ་རྗེས་\footnote{རྡོ་རྗེ་ཕྱག་རྒྱ་  སྣར་ཐང་། }སངས་རྒྱས་ལ། །མཆོད་ཆེན་ཇི་ལྟར་ཆེར་བྱིན་པ། །བདག་ལ་:བསྲུང་བའི་\footnote{སྲུང་བའི་  སྣར་ཐང་། }དོན་གྱི་ཕྱིར། །ནམ་མཁའ་རྡོ་རྗེ་བདག་ལ་\footnote{གསོལ་  སྣར་ཐང་།  པེ་ཅིན། }སྩོལ། །ཞེས་གསོལ་བ་བཏབ་ལ། དབང་བསྐུར་\footnote{སྐུར་  སྣར་ཐང་། }རྡོ་རྗེ་ཆེན་པོ་ནི། །ཁམས་གསུམ་ཀུན་གྱིས་ཕྱག་བྱས་པ། །སངས་རྒྱས་:ཀུན་གྱི་\footnote{ཀུན་གྱིས་  སྣར་ཐང་།  པེ་ཅིན། }གསང་:གསུམ་གྱི།\footnote{གསུམ་གྱིས།  སྣར་ཐང་། } །གནས་ལས་བྱུང་བ་སྦྱིན་པར་བྱ། །འདི་ལྟར་བརྗོད་པ་རྣམས་ཀྱིས་སངས་རྒྱས་ཀྱི་བཀས་སྤྱན་མ་ལ་སོགས་པས་དབང་བསྐུར་བས། དབུ་རྒྱན་དུ་སྣང་བ་མཐའ་ཡས་:བསམ་པར་\footnote{བསམས་པར་  སྣར་ཐང་། }བྱའོ། །དེ་ལྟར་བསྒོམས་ལ་བཟླས་པ་བྱས་ཏེ། སྐྱེན་པར་མ་ཡིན། བུལ་བར་མ་ཡིན། མི་གསལ་བར་མ་ཡིན། ཕྱི་མོ་དམན་པར་མ་ཡིན། ཡང་དག་པ་མ་ཡིན་པའི་རྟོག་པ་སྤངས་ཏེ། ཨོཾ་ཀུ་རུ་ཀུལླེ་ཧྲཱིཿ་ཧཱུཾ་སྭཱ་ཧཱ། དེ་ལྟར་ཐུན་མཚམས་གསུམ་གྱི་རིམ་\footnote{གོ་རིམས་  སྣར་ཐང་། }པས། ཁུར་བ་དང་། ཤ་ཀ་ར་ལ་སོགས་པའི་གཏོར་མ་སྦྱིན་པ་སྔོན་དུ་འགྲོ་བས། གུས་པས་ཡུན་རིང་པོར་:བསམ་པ་\footnote{བསམ་གཏན་  སྣར་ཐང་། }བརྟན་པོས་ཟླ་བ་དྲུག་ཏུ་:བསྒོམ་པར་\footnote{བསྒོམས་པར་  སྣར་ཐང་། }:བྱས་ན་\footnote{བྱས་ནས་  སྣར་ཐང་། }གྲུབ་པའི་མཚན་མ་འབྱུང་བར་འགྱུར་རོ། །དེ་ནས་ཟླ་བ་དང་པོ་ལ་ནད་ལ་སོགས་པའི་གནོད་པ་བརྒྱ་ཉེ་བར་འོང་བར་འགྱུར་རོ། །གཉིས་པ་ལ་སྟོན་མོ་ཡིད་དུ་འོང་བ་མ་བསླངས་པར་རྙེད་པར་འགྱུར་རོ། །གསུམ་པ་ལ་གོས་སྣ་ཚོགས་རྙེད་པ་དང་། བཞི་པ་ལ་སྣ་ཚོགས་ཀྱི་སོ་རྩི་རྒྱུན་མི་འཆད་པར་རྙེད་པར་འགྱུར་བ་དང་། ལྔ་པ་ལ་རྣམ་པ་སྣ་ཚོགས་པའི་གཟུགས་ལ་སོགས་པའི་ཡོན་ཏན་གྱིས་བརྒྱན་པའི་བུད་མེད་ཀྱི་ཚོགས་དང་ཕྲད་པ་རྙེད་པར་འགྱུར་རོ། །\footnote{། །ཟླ་བ་  སྣར་ཐང་།  པེ་ཅིན། }དྲུག་པ་:ལ་ལྷ་དང་\footnote{ལ་  སྣར་ཐང་།  པེ་ཅིན། }ཀླུ་དང་དྲི་ཟ་ལ་སོགས་པ་\footnote{པའི་  སྣར་ཐང་།  པེ་ཅིན། }བུད་མེད་མཛེས་མ་དང་ཕྲད་པར་འགྱུར་རོ། །ཕལ་ཆེར་བཅོམ་ལྡན་འདས་མ་བསྒོམས་\footnote{བསྒོམ་  སྣར་ཐང་། }ན་བུད་མེད་རྣམས་:ཀྱིས་འཚོ་\footnote{ཀྱིས་འཚེ་  སྣར་ཐང་། ཀྱི་འཚེ་  པེ་ཅིན། }བར་བྱེད་དོ། །དེ་ལ་བདུད་ཀྱི་ལས་འདི་རྣམས་སྤངས་ཏེ། བཅོམ་ལྡན་:འདས་མ་སྒོམ་\footnote{འདས་མ་བསྒོམ་  སྣར་ཐང་། འདས་བསྒོམ་  པེ་ཅིན། }པར་བྱེད་པ་ལ་དངོས་སུ་འམ། རྨི་ལམ་དུ་སྟོན་པར་འགྱུར་ཏེ། རལ་གྲི་དང་། མིག་\footnote{སྨིག་  སྣར་ཐང་། }སྨན་དང་། རྐང་པ་བྱུག་པ་དང་། མི་སྣང་བ་དང་། རིལ་བུ་དང་། དངུལ་ཆུ་དང་། བཅུད་ཀྱིས་:ལེན་པ་\footnote{ལེན་  སྣར་ཐང་། }དང་། མཁའ་སྤྱོད་དང་། ས་འོག་ལ་སོགས་པའི་དངོས་གྲུབ་ལ་སོགས་པ་\footnote{པའི་དངོས་གྲུབ་  སྣར་ཐང་། }འདོད་པ་རྣམས་གྲུབ་པར་བྱེད་དོ། །གལ་ཏེ་བསྟན་པར་མ་གྱུར་ན་སྔར་བརྗོད་པའི་ཕྱག་རྒྱ་:བཅིང་སྟེ།\footnote{བཅིངས་ཏེ།  སྣར་ཐང་། } མཆོད་པ་དང་། གཏོར་མ་སྔོན་དུ་འགྲོ་བས། ཟླ་བ་:དྲུག་གི་\footnote{དྲུག་གིས་  སྣར་ཐང་། }བར་དུ་ཉའི་ནུབ་མོ་ཐོག་ཐག་ཏུ་བཟླས་པ་བྱའོ། །དེ་ནས་ཕྱག་རྒྱ་:འབར་ཏེ་\footnote{འབར་སྟེ་  སྣར་ཐང་། }འགྲུབ་པར་འགྱུར་རོ། །ཕྱག་རྒྱ་དེས་ཚངས་པ་དང་། དབང་པོ་དང་། ཉེ་བའི་དབང་པོ་དང་། དྲག་པོ་དང་། སྲེད་མེད་ཀྱི་བུ་ལ་སོགས་པ་བཀུག་ནས་འོངས་ཏེ། ཕོ་ཉར་གྱུར་ནས་:སྒྲུབ་པ་\footnote{གྲུབ་པ་  སྣར་ཐང་། }པོའི་འདོད་པ་རྫོགས་པར་བྱེད་དོ། །དེ་ནས་བཟུང་སྟེ་སྐྱེ་བ་དང་། རྒས་པ་དང་:། ན་བ་དང་། \footnote{།    སྣར་ཐང་།  པེ་ཅིན། }འཆི་བ་དང་བྲལ་ཏེ་:འཇིག་རྟེན་གྱི་\footnote{འཇིག་རྟེན་གྱིས་  སྣར་ཐང་། }ཁམས་གཞན་དུ་སོང་\footnote{སོང་སོང་  སྣར་ཐང་།  པེ་ཅིན། }ནས་དེ་བཞིན་གཤེགས་པ་རྣམས་མཐོང་སྟེ། ས་དང་གཟུངས་\footnote{གཟུགས་  སྣར་ཐང་། }ལ་སོགས་པ་ཐོབ་པར་འགྱུར་རོ། །འཕགས་མ་སྒྲོལ་མའི་རྟོག་པའི་རིམ་པ། ཀུ་རུ་ཀུལླེའི་སྒྲུབ་ཐབས་སློབ་དཔོན་འཕགས་པ་:ཀླུ་སྒྲུབ་\footnote{ཀླུ་གྲུབ་  སྣར་ཐང་། ཀླུའི་སྒྲུབ་  པེ་ཅིན། }ཀྱིས་མཛད་པ་རྫོགས་སོ། །པཎྜི་ཏ་དོན་ཡོད་རྡོ་རྗེ་དང་། :ཁམས་པ་ལོ་ཙཱ་\footnote{ཁམས་པ་ལོཙྪ་  སྣར་ཐང་། ཁམས་པ་ལོ་ཙ་  པེ་ཅིན། }བ་དགེ་སློང་བ་རིས་བསྒྱུར་བའོ།། །།
