\chapter{༄༅། །སྒྲོལ་མའི་རྟོག་པ་ལས་འབྱུང་བའི་ཚིག་ལྷུག་པའི་རིགས་བྱེད་མའི་སྒྲུབ་ཐབས།}༄༅༅། །རྒྱ་གར་སྐད་དུ། མུཀྟེ་ཀེ་ན་ཏཱ་རོདྦྷ་བ་ཀུ་རུ་ཀུལླེ་\footnote{ཀུ་རུ་ལླེ་  སྣར་ཐང་།  པེ་ཅིན། }སཱ་དྷ་ནཾ། བོད་སྐད་དུ། སྒྲོལ་མའི་རྟོག་པ་ལས་:བྱུང་བའི་\footnote{འབྱུང་བའི་  སྣར་ཐང་།  པེ་ཅིན། }ཚིག་ལྷུག་པའི་:རིགས་བྱེད་\footnote{རིག་བྱེད་  སྣར་ཐང་།  པེ་ཅིན། }མའི་སྒྲུབ་ཐབས། འཕགས་མ་རིགས་\footnote{རིག་  སྣར་ཐང་།  པེ་ཅིན། }བྱེད་མ་ལ་ཕྱག་འཚལ་ལོ། །གང་གིས་དྲན་པ་ཙམ་གྱིས་ནི། །ལྷ་དང་ལྷ་མིན་མིར་བཅས་རྣམས། །ངེས་པར་མངག་གཞུག་\footnote{ཞུགས་  སྣར་ཐང་།  པེ་ཅིན། }བྱེད་འགྱུར་བའི། །སྒྲོལ་མ་དེ་ལ་ཕྱག་འཚལ་ལོ། །:དང་པོ་\footnote{དང་པོར་  སྣར་ཐང་།  པེ་ཅིན། }རེ་ཞིག་སྔགས་པས་ཡིད་དང་མཐུན་\footnote{འཐུན་  སྣར་ཐང་།  པེ་ཅིན། }པའི་གནས་སུ་:བུད་མེད་ཀྱི་\footnote{བུད་མེད་ཀྱིས་  སྣར་ཐང་།  པེ་ཅིན། }ཚོགས་ལ་སོགས་པ་དང་བྲལ་བར་སྟན་བདེ་བ་ལ་འདུག་སྟེ། རང་གི་སྙིང་གར་དབྱངས་ཡིག་དང་པོ་ཡོངས་སུ་གྱུར་པ་ལས་ཟླ་བའི་དཀྱིལ་འཁོར་གྱི་སྟེང་དུ་འོད་ཟེར་དང་བཅས་པའི་:ཧྲཱིཿ་ཡིག་གི་\footnote{ཧྲཱིཿ་ཡིག་གིས་  སྣར་ཐང་།  པེ་ཅིན། }ས་བོན་ལས་བྱུང་བའི་འོད་ཟེར་དམར་པོའི་ཚོགས་ཀྱིས་འགྲོ་བ་རྣམས་གསལ་བར་མཛད་པ་སྔོན་དུ་འགྲོ་བས་ཀུ་རུ་ཀུལླེའི་རི་ལ་གནས་པའི་བཅོམ་ལྡན་འདས་མ་རིགས་བྱེད་མ་ཡང་དག་པར་བསྐུལ་ནས་སྤྱན་དྲངས་ཤིང་མདུན་གྱི་ནམ་མཁའ་ལ་ཡང་དག་པར་བསམས་\footnote{བསམ་  སྣར་ཐང་། }ཏེ། སྙིང་གའི་ས་བོན་གྱི་འོད་ཟེར་ལས་སྤྲུལ་པའི་མེ་ཏོག་དང་བདུག་སྤོས་ལ་སོགས་པ་སྣ་ཚོགས་པའི་:མཆོད་པས་མཆོད་\footnote{མཆོད་  སྣར་ཐང་། }པར་བྱའོ། །དེའི་རྗེས་ལ་ཕྱག་འཚལ་བ་དང་། སྡིག་པ་བཤགས་པ་དང་། བསོད་ནམས་ལ་རྗེས་སུ་ཡི་རང་བ་དང་། ཡོངས་སུ་བསྔོ་བ་དང་། གསུམ་ལ་སྐྱབས་སུ་འགྲོ་བ་དང་། བྱང་ཆུབ་ཏུ་སེམས་བསྐྱེད་པ་སྔོན་དུ་འགྲོ་བས། དཀོན་མཆོག་གསུམ་ལ་བདག་སྐྱབས་མཆི། །སྡིག་པ་ཐམས་ཅད་སོ་སོར་བཤགས། །འགྲོ་བའི་དགེ་ལ་རྗེས་ཡི་\footnote{ཡིད་  སྣར་ཐང་། }རང་། །སངས་རྒྱས་བྱང་ཆུབ་ཡིད་ཀྱིས་གཟུང་། །ཞེས་ལན་གསུམ་བརྗོད་པར་བྱའོ། །དེ་ནས་ཚད་མེད་པའི་གནས་བཞི་:བསྒོམ་པ་སྔོན་དུ་འགྲོ་བས། ཆོས་ཐམས་ཅད་སྟོང་པ་ཉིད་དུ་བསྒོམ་\footnote{བསྒོམ་  སྣར་ཐང་། }ཞིང་། སྔགས་འདིས་བྱིན་གྱིས་:བརླབ་པ་\footnote{བརླབས་པ་  སྣར་ཐང་།  པེ་ཅིན། }བརྗོད་པར་བྱའོ།\footnote{བྱ་སྟེ།  སྣར་ཐང་། } །ཨོཾ་ཤཱུ་ནྱ་ཏཱ་ཛྙཱ་ན་བཛྲ་སྭ་\footnote{སྭཱ་  སྣར་ཐང་། }བྷཱ་:ཝ་ཨཱཏྨ་ཀོ྅\footnote{བ་ཨད་མ་ཀོ་  སྣར་ཐང་། }ཧཾ། དེའི་རྗེས་ལ་འགྲོ་བ་སྡུག་བསྔལ་བ་མཐའ་དག་ལ་སྙིང་རྗེ་བས། རང་གི་སེམས་ཀྱི་ས་བོན་གྱི་གཟུགས་ལ་བལྟ་སྟེ། དེ་ཡོངས་སུ་:གྱུར་པ་ལ་ཨུཏྤ་ལ་དམར་པོའི་སྟེང་ན་དབྱངས་ཡིག་\footnote{གྱུར་པ་ལས་ཨུད་པ་དམར་པོའི་སྟེང་དུ་དབྱངས་  སྣར་ཐང་། གྱུར་པ་ལ་ཨུཏྤ་ལ་དམར་པོའི་སྟེང་ན་དབྱངས་  པེ་ཅིན། }དང་པོ་ཡོངས་སུ་གྱུར་པ་ལས་ཟླ་བ་ལ་གནས་པའི་:ས་བོན་གྱི་\footnote{ས་བོན་གྱིས་  སྣར་ཐང་། }འོད་ཟེར་:རྣམས་ཀྱིས་\footnote{རྣམས་ཀྱི་  སྣར་ཐང་། }སྔོན་བཞིན་དུ་འགྲོ་བ་རྣམས་གསལ་བར་མཛད་ནས། དེ་བཞིན་གཤེགས་པ་བྱེ་བ་མཐའ་ཡས་པ་ཡང་དག་པར་བསྐུལ་ཞིང་སྤྱན་དྲངས་ནས་མདུན་\footnote{ནང་  སྣར་ཐང་། }དུ་བསྒོམ་པར་བྱའོ། །དེ་:ནས་ཨུཏྤལ་\footnote{ལ་ཨུད་པ་  སྣར་ཐང་། }ལ་སོགས་པ་ཡོངས་སུ་གྱུར་པ་ལས་སྐུ་མདོག་དམར་མོ། པདྨ་དམར་པོ་དང་ཟླ་བའི་གདན་ལ་བཞུགས་པ། དར་དམར་པོའི་སྟོད་གཡོགས་:བྱས་པས་\footnote{བྱས་པ་  སྣར་ཐང་།  པེ་ཅིན། }འོད་ཟེར་དམར་པོ་དང་ལྡན་པ། རིན་པོ་ཆེ་དམར་པོའི་རྣ་རྒྱན་\footnote{བརྒྱན་  སྣར་ཐང་། }ཅན། ཕྱག་བཞི་པ། གཡས་པས་མི་འཇིགས་པ་རབ་ཏུ་སྦྱིན་པ། གཉིས་པས་མདའ་ཡང་དག་པར་འགེངས་པ། གཡོན་ན་རིན་པོ་ཆེའི་གཞུ་བསྣམས་\footnote{བསམས་  སྣར་ཐང་། }པ། གཉིས་པས་ཨུཏྤལ་\footnote{ཨུད་པ་ལ་  སྣར་ཐང་། }དམར་པོ་བསྣམས་པ། དེ་བཞིན་གཤེགས་པ་འོད་དཔག་མེད་ཀྱི་དབུ་རྒྱན་\footnote{བརྒྱན་  སྣར་ཐང་།  པེ་ཅིན། }ཅན། ཀུ་རུ་ཀུལླེའི་རིའི་ཕུག་ན་བཞུགས་པ། :སྒྲ་གཅན་ཟིན་\footnote{སྒྲ་གཅན་  སྣར་ཐང་། }གྱི་མགོ་ལ་གནས་པའི་འདོད་ལྷ་ཆུང་མ་དང་བཅས་པའི་སྟེང་དུ་བཞུགས་པ། སྒེག་པའི་ཉམས་ཀྱིས་རོལ་ཞིང་དང་པོའི་ལང་ཚོ་དང་ལྡན་པའི་:རིགས་བྱེད་\footnote{རིག་བྱེད་  སྣར་ཐང་།  པེ་ཅིན། }མ་བསྒོམ་པར་བྱའོ། །:དེ་ལྟར་ན་\footnote{དེ་ལྟར་  སྣར་ཐང་། }དམ་ཚིག་སེམས་དཔའ་\footnote{གསལ་  སྣར་ཐང་། }བསྐྱེད་ནས་སྐད་ཅིག་གིས་ཡེ་ཤེས་སེམས་དཔའ་མདུན་དུ་སྤྱན་དྲངས་ཏེ་ཡང་དག་པར་མཆོད་ནས་དམ་ཚིག་གི་ཕྱག་རྒྱས་:ཞལ་གྱི་ནང་ནས་\footnote{ཞལ་གྱིས་  སྣར་ཐང་། }ནང་དུ་གཞུག་\footnote{ཞུགས་  སྣར་ཐང་། }པར་\footnote{པར་སྒོམ་པར་  སྣར་ཐང་། པར་བསྒོམ་པར་  པེ་ཅིན། }བྱའོ། །དེ་ལ་དམ་ཚིག་གི་ཕྱག་རྒྱ་ནི་འདི་ཡིན་ཏེ། ཐལ་མོ་ཡང་དག་པར་སྦྱར་ནས་གུང་\footnote{གང་  སྣར་ཐང་། }མོ་དག་རྩེ་མོར་བྱས་པའི་དབུས་སུ་མིང་མེད་དག་བཅུག་སྟེ། གུང་མོའི་རྒྱབ་ཏུ་མཛུབ་\footnote{འཛུབ་  སྣར་ཐང་། }མོ་གཉིས་གནས་ཤིང་མཐེབ་ཆེན་གཉིས་འོག་:ཏུ་གཞུག་\footnote{ཏུ་རབ་ཏུ་ཞུགས་  སྣར་ཐང་། ཏུ་རབ་ཞུགས་  པེ་ཅིན། }པ་དེ་ལྟ་བུའི་རྣམ་པ་ནི་ཨུཏྤ་\footnote{ཨུད་པ་  སྣར་ཐང་། }ལའི་ཕྱག་རྒྱའོ། །དེའི་རྗེས་སུ། བྱང་ཆུབ་:རྡོ་རྗེས་\footnote{རྡོ་རྗེ་  སྣར་ཐང་།  པེ་ཅིན། }སངས་རྒྱས་ལ། །ཇི་ལྟར་མཆོད་ཆེན་སྩལ་བ་ལྟར། །བདག་ཀྱང་བསྐྱབ་པའི་དོན་གྱི་ཕྱིར། །ནམ་མཁའི་རྡོ་རྗེ་དེང་བདག་སྩོལ།\footnote{གསོལ།  སྣར་ཐང་།  པེ་ཅིན། } །ཞེས་གསོལ་བ་བཏབ་ལ། དབང་བསྐུར་བ་ནི་རྡོ་རྗེ་ཆེ། །ཁམས་གསུམ་ཀུན་གྱིས་ཕྱག་བྱས་པ། །སངས་རྒྱས་:ཀུན་གྱི་\footnote{ཀུན་གྱིས་  སྣར་ཐང་། }གསང་གསུམ་གྱི། །གནས་ལས་བྱུང་བ་བདག་གིས་སྦྱིན། །ཞེས་བརྗོད་པ་:རྣམས་ཀྱི་\footnote{རྣམས་ཀྱིས་  སྣར་ཐང་།  པེ་ཅིན། }སངས་རྒྱས་ཀྱི་བཀས་\footnote{བཀའ་  སྣར་ཐང་། }སྤྱན་མ་ལ་སོགས་པ་རྣམས་ཀྱིས་དབང་བསྐུར་བ་སྟེར་\footnote{བསྟེར་  སྣར་ཐང་། }བར་བསམ་པར་བྱ་ཞིང་འོད་དཔག་མེད་དབུ་རྒྱན་དུ་བཞུགས་པའོ། །དེ་ལྟར་བསྒོམས་ནས་:བཟླས་པ་\footnote{བསླབས་པར་  སྣར་ཐང་། }བྱ་སྟེ། ཧ་ཅང་སྐྱེན་:མིན་བུལ་བའང་མིན།\footnote{མིན།  སྣར་ཐང་། } །མི་:གསལ་མ་ཡིན་ཆ་མཉམ་མིན། །ཡང་དག་མ་\footnote{གསལ་མ་  སྣར་ཐང་། }ཡིན་རྟོག་པ་སྤངས། ཨོཾ་ཀུ་རུ་ཀུལླེ་\footnote{ཀུ་ལུ་  སྣར་ཐང་། }ཧྲཱིཿ་ཧཱུཾ་སྭཱ་ཧཱ། དེ་ལྟར་ཐུན་\footnote{མཐུན་  སྣར་ཐང་། }མཚམས་གསུམ་གྱི་:རིམ་པས་\footnote{རིམས་པས་  སྣར་ཐང་། }:ཁུར་བ་\footnote{འཁུར་བ་  སྣར་ཐང་། }དང་:ཀ་ར་\footnote{ཀ་ར་བ་  སྣར་ཐང་། }ལ་སོགས་པའི་གཏོར་མ་དབུལ་བ་སྔོན་དུ་:འགྲོ་བས་\footnote{འཕྲོ་བས་  སྣར་ཐང་། }རྟག་ཏུ་རྒྱུན་མ་ཆད་པར་བརྟན་པར་འཇུག་པས་ཟླ་བ་དྲུག་ཏུ་:བསྒོམ་པར་\footnote{སྒོམ་པར་  སྣར་ཐང་། }བྱའོ། །གྲུབ་པའི་མཚན་མ་རྣམས་ནི་འབྱུང་བར་འགྱུར་རོ། །དེའི་རྗེས་ལ་ཟླ་བ་:དང་པོ་\footnote{དང་བ་  སྣར་ཐང་། }ལ་ནད་ལ་སོགས་པ་བར་དུ་གཅོད་པ་བརྒྱ་འབྱུང་བར་འགྱུར་རོ། །ཟླ་བ་གཉིས་པ་ལ་:སྟོན་མོ་གྱ་\footnote{སྟོན་མོ་གྱི་  སྣར་ཐང་། }ནོམ་པ་མ་བསླངས་པར་:རྙེད་པར་\footnote{སྙེད་པར་  སྣར་ཐང་། }:འགྱུར་རོ།\footnote{འགྲུབ་བོ།  སྣར་ཐང་། } །གསུམ་པ་ལ་གོས་སྣ་ཚོགས་པ་རྙེད་པར་འགྱུར་རོ། །བཞི་པ་ལ་སྣ་ཚོགས་པའི་སོ་རྩི་རྒྱུན་མི་འཆད་པར་རྙེད་པར་འགྱུར་རོ། །ལྔ་པ་ལ་རྣམ་པ་སྣ་ཚོགས་པའི་གཟུགས་ལ་སོགས་པའི་ཡོན་ཏན་གྱིས་བརྒྱན་པའི་:བུད་མེད་རྣམས་ཀྱིས་\footnote{བུད་མེད་བྱི་  སྣར་ཐང་། }གྲོགས་རྙེད་པར་འགྱུར་རོ། །ཟླ་བ་དྲུག་པ་ལ་ལྷ་དང་ཀླུ་དང་། གནོད་སྦྱིན་དང་། དྲི་ཟ་དང་། མིའམ་ཅི་ལ་སོགས་པའི་བུད་མེད་ཡིད་དུ་འོང་བ་དང་ཕྲད་པར་འགྱུར་རོ། །བཅོམ་ལྡན་འདས་མ་བསྒོམས་ན་ཕལ་ཆེར་བུད་མེད་རྣམས་ཀྱིས་འཚོ་བར་འགྱུར་རོ། །དེ་ལ་བདུད་ཀྱི་ལས་འདི་རྣམས་སྤངས་ཏེ། བཅོམ་ལྡན་འདས་མ་སྒོམ་པར་བྱེད་པ་ལ་རྨི་ལམ་མམ་དངོས་སུ་སྟོན་པར་འགྱུར་རོ། །རལ་གྲི་དང་། མིག་སྨན་དང་། རྐང་པའི་:བྱུག་པ་\footnote{ཕྱུག་པ་  སྣར་ཐང་། }དང་། མི་སྣང་བ་དང་། བཅུད་ལེན་དང་། མཁའ་ལ་སྤྱོད་པ་དང་། ས་ལ་སྤྱོད་པ་དང་། ས་འོག་གི་དངོས་གྲུབ་ལ་སོགས་པའི་དངོས་གྲུབ་རྣམས་བསྒྲུབ་པར་བྱའོ། །གལ་ཏེ་སོ་སོར་བཀས་སྟོན་པར་:མ་གྱུར་\footnote{མི་འགྱུར་  སྣར་ཐང་། }ན། དེའི་ཚེ་སྔར་གསུངས་པའི་ཕྱག་རྒྱ་བཅིངས་ནས་མཆོད་པ་དང་། གཏོར་མ་དབུལ་བ་སྔོན་དུ་འགྲོ་བས་ཟླ་བ་དྲུག་གི་མཐར་ཟླ་བ་ཡོངས་སུ་གང་བའི་མཚན་མོ་:རིལ་པོར་\footnote{རིལ་པོ་  སྣར་ཐང་། }བཟླས་པར་བྱའོ། །དེ་ནས་ཕྱག་རྒྱ་འབར་\footnote{འདས་  སྣར་ཐང་། }ཏེ་འགྲུབ་པར་འགྱུར་རོ། །ཕྱག་རྒྱ་དེས་ཚངས་པ་དང་། དབང་པོ་དང་། ཉེ་བའི་དབང་པོ་དང་། དྲག་པོ་དང་། སྲེད་\footnote{སྲིད་  སྣར་ཐང་། }མེད་ཀྱི་བུ་ལ་སོགས་པ་ཡང་དག་པར་བཀུག་སྟེ། ཡང་དག་པར་འོངས་ནས་མངོན་པར་འདོད་པ་རྣམས་རྫོགས་པར་སྒྲུབ་པར་བྱེད་པར་འགྱུར་རོ། །དེ་ནས་བཟུང་སྟེ་སྐྱེ་བ་དང་:།རྒས་པ་དང་། \footnote{།   སྣར་ཐང་། }འཆི་བ་དང་བྲལ་ཞིང་གྲུབ་པའི་འཇིག་རྟེན་གྱི་:ཁམས་རྣམས་\footnote{ཁམས་  སྣར་ཐང་། }སུ་སོང་ནས་དེ་བཞིན་གཤེགས་པ་རྣམས་བལྟ་ཞིང་ས་དང་གཟུངས་ལ་སོགས་པ་རྣམས་ཐོབ་པར་འགྱུར་རོ། །སྒྲོལ་མའི་རྟོག་པ་ལས་བྱུང་བའི་ཚིག་ལྷུག་པའི་ཀུ་\footnote{སྐུ་  སྣར་ཐང་། }རུ་ཀུལླེའི་སྒྲུབ་ཐབས་སློབ་དཔོན་འཕགས་པ་ཀླུ་སྒྲུབ་ཀྱིས་མཛད་པ་རྫོགས་སོ།། །།
