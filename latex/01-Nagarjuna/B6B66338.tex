\chapter{༄༅། །འཇམ་དཔལ་རྡོ་རྗེ་བློ་གྲོས་འཕེལ་བ།}༄༅༅། །རྒྱ་གར་སྐད་དུ། མཉྫུ་ཤྲཱི་བཛྲ་པྲཛྙཱ་བརྡྷ་:ན་ནཱ་\footnote{ནི་སྭཱ་ཧཱ་ན་  པེ་ཅིན། }མ། བོད་སྐད་དུ། འཇམ་དཔལ་རྡོ་རྗེ་བློ་གྲོས་:འཕེལ་བ་\footnote{འཕེལ་  སྣར་ཐང་།  པེ་ཅིན། }ཞེས་བྱ་བ། བཅོམ་ལྡན་འདས་འཇམ་པའི་\footnote{དཔལ་  སྣར་ཐང་། }མགོན་ལ་ཕྱག་འཚལ་ལོ། །ཤེས་རབ་འཕེལ་བར་འདོད་པས་གནས་ཁང་དབེན་ཞིང་ཉམས་དགའ་བར་གཙང་བ་དྲི་ཞིམ་པོའི་ཆུས་ཆག་ཆག་བཏབ་ལ། མཎྜལ་ཁྲུ་:གང་བ་ཅིག་\footnote{གང་པ་ཞིག་  སྣར་ཐང་། གང་པ་གཅིག་  པེ་ཅིན། }བྱས་ཏེ། དེའི་དབུས་སུ་པདྨ་འདབ་མ་བརྒྱད་པ་སེར་པོ་དབུས་དཀར་པོར་བྱས་ལ། དེའི་སྟེང་དུ་ལྷུང་བཟེད་དུ་ཤེས་རབ་འཕེལ་བར་བྱེད་པའི་རྫས། དབང་པོ་ལག་པ།\footnote{ན།  སྣར་ཐང་། } གླ་རྩི། གི་ཝང་། སྤང་རྒྱན་དཀར་པོ། ཙནྡན་དཀར་པོ་རྣམས་བ་:དམར་པོའི་\footnote{དམར་མོའི་  སྣར་ཐང་།  པེ་ཅིན། }འོ་མས་སྦྲུས་ཏེ། རིལ་བུ་སྲན་མ་ཙམ་བྱའོ། །དེའི་སྟེང་དུ་འཇམ་དཔལ་དང་དབྱངས་ཅན་མ་གཉིས་ཞལ་སྦྱོར་གྱི་ཙ་:ཀ་ལི་ཅིག་\footnote{ཀ་ལི་གཅིག་  སྣར་ཐང་།  པེ་ཅིན། }བཞུགས་པར་བྱའོ། །དེ་ནས་ཨོཾ་སྭ་\footnote{སྭཱ་  སྣར་ཐང་།  པེ་ཅིན། }བྷཱ་བ་:བི་ཤུདྡྡྷཿ་སརྦ་དྷརྨཱཿ་སྭ་བྷཱ་\footnote{བི་ཤུདྡྷཿསརྦ་དྷརྨཱཿསྭ་བྷཱ་  སྣར་ཐང་། བི་ཤུད་དྷ་སརྦ་དྷརྨཱ་སྭཱ་བྷ་  པེ་ཅིན། }བ་བི་ཤུདྡྷོ྅\footnote{ཤུད་དྷོ་  པེ་ཅིན། }ཧཾ་གི་སྔགས་ཀྱིས་ཆོས་ཐམས་ཅད་སྟོང་པ་ཉིད་དུ་བསྒོམ་པར་བྱའོ། །དེའི་ངང་ལས་པདྨ་དམར་པོ་འདབ་མ་:བརྒྱད་པ་ཅིག་\footnote{བརྒྱད་པ་ཞིག་  སྣར་ཐང་། བརྒྱད་པ་གཅིག་  པེ་ཅིན། }བསམ། དེའི་སྟེང་དུ་ཨ་ལས་ཟླ་བའི་དཀྱིལ་འཁོར་གྱི་སྟེང་དུ་དྷཱིཿ་\footnote{དྷིཿ་  སྣར་ཐང་། དྷི་  པེ་ཅིན། }སེར་པོ་གུར་གུམ་གྱི་མདོག་:ཅན་ཅིག་\footnote{ཅན་ཞིག་  སྣར་ཐང་། ཅན་གཅིག་  པེ་ཅིན། }བསམ། དེ་ཡོངས་སུ་གྱུར་པ་ལས་བདག་ཉིད་འཕགས་པ་འཇམ་དཔལ་སྐུ་མདོག་དམར་པོ་ཕྱག་གཡས་ན་ཤེས་རབ་ཀྱི་རལ་གྲི་གཡོན་ན་ཤེས་རབ་ཀྱི་པུ་སྟི་བསྣམས་པ། རིན་པོ་ཆེའི་རྒྱན་སྣ་ཚོགས་ཀྱིས་བརྒྱན་པ། ཞལ་འཛུམ་བག་དང་ལྡན་པ་བསྒོམ་པར་བྱའོ། །མདུན་གྱི་ལྷུང་བཟེད་ཀྱི་ནང་དུ་ཧྲཱིཿ་\footnote{ཧྲི་  པེ་ཅིན། }དཀར་པོ་ཡོངས་སུ་གྱུར་པ་ལས་པདྨ་དང་ཟླ་བའི་སྟེང་དུ། :དབྱངས་ཅན་མ་\footnote{དབྱངས་ཅན་  ཅོ་ནེ། }དཀར་མོ་གཡས་ན་རལ་གྲི་གཡོན་ན་པུ་སྟི་བསྣམས་པ། རིན་པོ་ཆེའི་རྒྱན་དང་བཅས་པ་ཞལ་འཛུམ་བག་དང་ལྡན་པ་དག་བསྒོམ་པར་བྱའོ། །དེ་ནས་ས་བོན་གཉིས་ལས་འོད་འཕྲོས་པས་རང་བཞིན་གྱི་གནས་ནས། ཡེ་ཤེས་སེམས་དཔའ་སྤྱན་དྲངས་ནས་སོ་སོར་བསྟིམ་མོ། །ཡང་\footnote{ཡང་ན་  སྣར་ཐང་། }འོད་ཟེར་འཕྲོས་པས་སངས་རྒྱས་དང་བྱང་ཆུབ་སེམས་དཔའ་རྣམས་ཀྱི་ཤེས་རབ་དང་མཐུ་དང་ནུས་པ་ཐམས་ཅད་བསྡུས་ནས་རང་དང་རྫས་ལ་བསྟིམ་པར་བྱའོ། །དེ་ནས་བཟླས་པ་བྱ་སྟེ། ན་མོ་མཉྫུ་:ཤྲཱི་བཱ་གཱི་\footnote{ཤྲཱི་བ་གི་  པེ་ཅིན། }ཤྭ་རི་མུཾ།\footnote{མུ།  ཅོ་ནེ།  པེ་ཅིན། } :བཱ་གཱི་ཤྭ་\footnote{བཱ་གི་ཤ་  སྣར་ཐང་། བ་གི་ཤྭ་  པེ་ཅིན། }རི་ཧོཿ།\footnote{ཧོ།  སྣར་ཐང་།  པེ་ཅིན། } ཨུ་པ་དེ་ཤ་པྲ་མོ་ད་ཏ་དྷ་བ་ནི་ཧྲཱིཿ་\footnote{ཧྲི་  པེ་ཅིན། }སྭཱ་ཧཱ། སྔགས་ཀྱི་འོད་ཟེར་བདག་གི་ཞལ་ནས་ལྷ་མོའི་:ཞལ་དུ། \footnote{ཞལ་  པེ་ཅིན། }ལྷ་མོའི་ཞལ་ནས་བདག་གི་ཐུགས་ཀར་ཐིམ་པར་བསྒོམ། དེ་ལས་འོད་ཟེར་འཕྲོས་པས་སེམས་ཅན་ཐམས་ཅད་ཀྱི་དོན་དུ་ཤེས་རབ་ཀྱི་མི་མཐུན་པའི་ཕྱོགས། གཏི་མུག་ལ་སོགས་པ་ཐམས་ཅད་བསལ་བར་བསྒོམ། འཇིག་རྟེན་དྲག་བྱེད་ཀྱི་པྲཛྙཱ་ཐམས་ཅད་བསྡུས་པར་བསྒོམ། སྨན་གྱི་རི་ལུ་དང་བདག་ལ་ཐིམ་པས་ཇི་ལྟར་འདོད་པའི་མཚན་ཉིད་ལ་སོགས་པའི་དངོས་གྲུབ་ཐམས་ཅད་ཐོབ་པར་བསྒོམ་མོ། །འདིའི་ཕན་ཡོན་ནི་ཉིན་གཅིག་བསྒྲུབས་ན་ཉིན་གཅིག་ཚིགས་སུ་བཅད་པ་སྟོང་ཟིན་པར་འགྱུར་རོ། །ཉི་མ་གཉིས་བསྒྲུབས་\footnote{སྒྲུབས་  སྣར་ཐང་། སྒྲུབ་  པེ་ཅིན། }ན་བསྟན་བཅོས་ཐམས་ཅད་ལ་མཁས་པར་འགྱུར་རོ། །:ཉི་མ་གསུམ་བསྒྲུབས་ན་སྐྱེ་བ་དྲན་པར་འགྱུར་རོ། །ཉི་མ་\footnote{ཉི་མ་  པེ་ཅིན། }བཞི་ཡིས་སངས་རྒྱས་དང་པྲ་ཛྙཱ་\footnote{པྲཱཛྙཱ་  པེ་ཅིན། }མཉམ་པར་:འགྱུར་རོ། །\footnote{འགྱུར་  སྣར་ཐང་། }ལྔ་ཡིས་མ་བསླབས་པའི་ཆོས་ལ་མཁས་པར་འགྱུར་རོ། །དྲུག་གིས་འཇམ་དཔལ་གྱི་ཞལ་མཐོང་བར་འགྱུར་རོ། །བདུན་གྱིས་གཞན་གྱི་སེམས་ཤེས་པར་འགྱུར་རོ། །སྒྲུབ་པའི་ཚད་ཞག་བདུན་དུ་བྱའོ། །རྟགས་ནི་འོད་དང་སྒྲ་ལ་སོགས་པ་འབྱུང་ངོ། །ཞག་བརྒྱད་ན་དཀྱིལ་འཁོར་གྱི་ལྷ་རང་དང་རི་ལུ་ལ་བསྟིམ་རིལ་བུ་དེ་སྒྲུབ་པ་པོས་:ཟོས་པས།\footnote{ཟོས་པ།  པེ་ཅིན། } འཇམ་པའི་དབྱངས་དང་ཤེས་རབ་མཉམ་པར་འགྱུར་རོ། །ཐུན་མོང་ཚེའི་དངོས་གྲུབ་ཀྱང་འཐོབ་སྟེ། དེ་ལྟར་ཡང་། འདི་ཟོས་ཡོན་ཏན་བསམ་མི་ཁྱབ། །ལོ་བརྒྱ་ཐུབ་པ་བརྒྱད་བརྒྱར་འགྲོ། །དགུ་བཅུ་ལོན་པ་བཅུ་དྲུག་:ལྟ་བུ་\footnote{ལྟ་བུར་  སྣར་ཐང་།  པེ་ཅིན། }འགྱུར། །སྐྲ་དཀར་བུད་ནས་ནག་པོར་འགྱུར། །བང་ནི་རྟ་དང་མཉམ་པར་འགྱུར།
