\chapter{༄༅། །དཔལ་མགོན་པོའི་གཏོར་མ།}༄༅༅། །རྒྱ་གར་སྐད་དུ། མ་ཧཱ་ཀཱ་ལ་བ་:ལིཾ་ཏེ་\footnote{ལིཾ་ཏ་  སྣར་ཐང་།  པེ་ཅིན། }སྱ་:ནཱ་མ།\footnote{ནཱ་མ་།  སྣར་ཐང་།  པེ་ཅིན། } བོད་སྐད་དུ། :དཔལ་མགོན་པོའི་\footnote{དཔལ་མགོན་པོ་ནག་པོའི་  སྣར་ཐང་།  པེ་ཅིན། }གཏོར་མ་ཞེས་བྱ་བ། དཔལ་ལྕམ་དྲལ་ལ་ཕྱག་འཚལ་ལོ། །དུར་ཁྲོད་:ལ་སོགས་པའི་\footnote{ལ་སོགས་པ་  སྣར་ཐང་།  པེ་ཅིན། }ཡིད་དུ་འོང་བའི་གནས་སུ། :བྱང་ཆུབ་ཀྱི་\footnote{བྱང་ཆུབ་ཀྱིས་  སྣར་ཐང་།  པེ་ཅིན། }སེམས་སྔོན་དུ་བཏང་ལ། བམ་བསྒྲུབ་བམ་གཏོར་མ་དབུལ་ཏེ། གཏོར་མ་ལ་གསུམ། བདག་ཁྱད་པར་དུ་བྱ་བ་དང་། གཏོར་མ་ཁྱད་པར་དུ་བྱ་བ་དང་། མགྲོན་ཁྱད་པར་དུ་བྱ་བའོ། །དང་པོ་ནི་:རང་ཉིད་\footnote{རང་  སྣར་ཐང་།  པེ་ཅིན། }སྐད་ཅིག་ལ་བཅོམ་ལྡན་འདས་མི་གཡོ་བར་བསྒོམས་ལ། སྐུ་གསུང་ཐུགས་ཀྱི་:ཡེ་ཤེས་\footnote{ཡི་གེས་  སྣར་ཐང་།  པེ་ཅིན། }བྱིན་གྱིས་བརླབ་པར་བྱའོ། །གཉིས་པ་ནི་བནྡྷ་འམ་ཐོད་པའི་ནང་དུ་མིའི་ཤ་ཆེན་ནམ་ཤ་ཁྲག་གིས་བརྒྱན་པའི་མགོན་པོའི་གཏོར་མ་བཟང་པོ་བཤམས།\footnote{བཤམ།  སྣར་ཐང་།  པེ་ཅིན། } དེ་མེད་ན་སྣོད་གཞན་དུ་མར་གྱིས་བརྒྱན་པ་བཤམ་མོ། །གཏོར་མ་བཤམས་ཟིན་པ་དང་། ཨོཾ་སྭ་བྷཱ་ཝ་:བི་ཤུདྡྷཿ་སརྦ་དྷརྨཱཿ་སྭ་བྷཱ་ཝ་བི་ཤུདྡྷོ྅ཧཾ་ཞེས་བྱ་བ་\footnote{བི་ཤུདྡྷཿ་སརྦ་དྷརྨཱཿ་སྭ་བྷཱ་ཝ་བི་ཧཾ་ཞེས་པ་  སྣར་ཐང་།  པེ་ཅིན། }ལན་གསུམ་བརྗོད་པས་གཏོར་མ་མི་དམིགས་པར་སྟོང་པར་བསྒོམ། སྟོང་པའི་ངང་ལས་ཡཾ་:སྔོན་པོ་ཅིག་\footnote{སྔོན་པོ་ཞིག་  སྣར་ཐང་།  པེ་ཅིན། }ཡོངས་སུ་གྱུར་པ་ལས། རླུང་གི་དཀྱིལ་འཁོར་ཁ་དོག་སྔོ་སྐྱར་འདུག་པའི་དབྱིབས་གཞུ་བཞིན་ནར་མོར་འདུག་པ། གཞུ་མཆོག་གཉིས་:ནི་མདའ་\footnote{ན་མར་  སྣར་ཐང་།  པེ་ཅིན། }དར་གྱི་བ་དན་གྱིས་མཚན་པ་:ཅིག་བསྒོམ།\footnote{ཞིག་བསྒོམས།  སྣར་ཐང་།  པེ་ཅིན། } དེའི་སྟེང་དུ་རཾ་ལས་མེའི་དཀྱིལ་འཁོར་གྲུ་གསུམ་འབར་བ་:ཚ་བ་ཅིག་\footnote{ཚ་བ་ཞིག་  སྣར་ཐང་།  པེ་ཅིན། }བསྒོམ། དེའི་:སྟེང་ན་\footnote{སྟེང་དུ་  སྣར་ཐང་།  པེ་ཅིན། }ཨ་\footnote{ཨ་ལས་  སྣར་ཐང་།  པེ་ཅིན། }ཡོངས་སུ་གྱུར་པ་ལས་ཐོད་པ་ཤིན་ཏུ་ཡངས་ཤིང་:ཆེ་བ་ཅིག་\footnote{ཆེ་བ་གཅིག་  སྣར་ཐང་།  པེ་ཅིན། }བསམ་མོ། །དེའི་སྟེང་དུ་:ཧྲཱིཿ་དཀར་\footnote{ཧྲི་དཀར་  སྣར་ཐང་།  པེ་ཅིན། }པོ་དེ་ལས་འོད་ཟེར་འཕྲོས་གཏོར་མ་ལ་མར་\footnote{མར་ལ་  སྣར་ཐང་།  པེ་ཅིན། }འཕྲོས་སོང་བས། གཏོར་མའི་མི་གཙང་བ་དང་། སྣོད་ཀྱི་དྲི་མ་དག་པར་བསམ་མོ།\footnote{མོས།  སྣར་ཐང་། } །དེར་ཧྲཱིཿ་ཡང་བདུད་རྩིར་ཞུ་བར་བསམ། དེ་ནས་གཏོར་མ་བས་ཁྲུ་གང་གིས་མཐོ་བའི་སྟེང་དུ་རཾ་ལས་ཉི་མའི་དཀྱིལ་འཁོར་དེའི་སྟེང་དུ་ཧཱུཾ་\footnote{ཧཱུཾ་་  སྣར་ཐང་། }མཐིང་ནག་དེ་:ཡོངས་སུ་\footnote{ཡོངས་  པེ་ཅིན། }གྱུར་པ་ལས་རྡོ་རྗེ་ནག་པོ་རྩེ་དགུ་པ་ཁོང་སེང་:ཅན་ཅིག་\footnote{ཅན་གཅིག་  སྣར་ཐང་།  པེ་ཅིན། }བསམ། དེའི་སྟེང་དུ་རཾ་ཡོངས་སུ་གྱུར་པ་ལས་ཉི་མའི་དཀྱིལ་འཁོར། དེའི་སྟེང་དུ་ཧཱུཾ་མཐིང་ནག་ཅིག་བཞེངས་ནས་བཞུགས་པར་བསམ། ཧཱུཾ་ལས་འོད་ཟེར་འཕྲོས་ནས་རྡོ་རྗེ་ཁོང་སེང་ནས་སོང་། རྡོ་རྗེའི་ར་\footnote{རྭ་  སྣར་ཐང་།  པེ་ཅིན། }འོག་མ་ནས་གཉིས་སུ་གྱེས་ནས་སོང་བས། གཞུ་མཆོག་གཉིས་ཀྱིས་བ་དན་ལ་བརྒྱབ་པས། བ་དན་གྱིས་:རླུང་གཡབ་པས་\footnote{རླུང་གཡབ་  སྣར་ཐང་།  པེ་ཅིན། }རླུང་གིས་མེ་:སྦར་མེས་\footnote{འབར་མེས་  སྣར་ཐང་།  པེ་ཅིན། }ཐོད་པ་ཁོལ། འོད་ཡར་:འཕྲོས་ནས་\footnote{འཕྲོས་  སྣར་ཐང་།  པེ་ཅིན། }སོང་བས། སངས་རྒྱས་དང་བྱང་ཆུབ་སེམས་དཔའ་དང་དེ་བཞིན་གཤེགས་པའི་ཐུགས་ཀ་ནས་ཡེ་ཤེས་ཀྱི་བདུད་རྩི་རྣམ་པ་ལྔ་\footnote{ལྔའི་  སྣར་ཐང་།  པེ་ཅིན། }ཆར་འབབ་པར་བསམས་ལ། ངག་ཏུ་ཨོཾ་ཨ་ཀཱ་རོ་མུ་ཁཾ་སརྦ་དྷརྨཱ་ཎཱཾ་ཨཱདྱ་ནུ་ཏྤནྣ་ཏྭཱཏ་ཨོཾ་ཨཱཿ་ཧཱུཾ་ཕཊ་སྭཱ་ཧཱ་ཞེས་བྱ་བའི་སྔགས་ལན་ལྔའམ་བདུན་བརྗོད། ཨཱ་\footnote{ཨཱ་་  སྣར་ཐང་།  པེ་ཅིན། }ཧཱུཾ་དང་དེ་བཞིན་བརྗོད་རྡོ་རྗེའི་སྟེང་གི་ཉི་མ་དང་། འོག་གི་ཉི་མ་གཉིས་ཧཱུཾ་དང་གསུམ་:འཁྱག་པའི་\footnote{འཁྱགས་པའི་  སྣར་ཐང་།  པེ་ཅིན། }ཟར་བུ་ཞུ་བ་གཙུག་\footnote{ཙུག་  སྣར་ཐང་།  པེ་ཅིན། }ཏུ་བདུད་རྩིར་ཞུ་བར་བསམས་ལ། དེ་ནས་རང་གི་ལག་པས་རྡོ་རྗེ་བླངས་ནས་ལན་གསུམ་དཀྲུགས་པས་བདུད་རྩི་དཀར་ཆིལ་གྱུར་པར་བསམ་མོ། །དེ་ནས་རང་གི་ལག་པའི་རྡོ་རྗེ་མི་དམིགས་པར་:བསྒོམ་མོ།\footnote{བསྒོམ།  སྣར་ཐང་།  པེ་ཅིན། } །དེ་ནི་གཏོར་མ་ཁྱད་པར་དུ་:བྱ་བའོ།\footnote{བྱའོ།  སྣར་ཐང་།  པེ་ཅིན། } །མགྲོན་ཁྱད་པར་དུ་བྱ་བ་ནི་གཏོར་མའི་མདུན་གྱི་ནམ་མཁའ་ལ་རཾ་ཡོངས་སུ་གྱུར་པ་ལས་ཉི་མའི་དཀྱིལ་འཁོར། དེའི་:སྟེང་དུ་\footnote{སྟེང་ན་  སྣར་ཐང་། སྟེང་ལ་  པེ་ཅིན། }ཧཱུཾ་:མཐིང་ནག་ཅིག་\footnote{མཐིང་ནག་གཅིག་  སྣར་ཐང་།  པེ་ཅིན། }བསམ། དེ་ལས་འོད་འཕྲོས་སེམས་ཅན་ཐམས་ཅད་ལ་ཕོག །སྡིག་སྒྲིབ་དག །ཐམས་ཅད་དཔལ་མགོན་པོའི་སྐུར་གྱུར། དེ་སླར་འདུས་པས་དཔལ་མགོན་པོ་སྐུ་མདོག་མཐིང་ནག་ཞལ་གཅིག་ཕྱག་གཉིས་པ་:གཡས་གཡོན་ན་\footnote{གཡས་གཡོན་  སྣར་ཐང་།  པེ་ཅིན། }མི་ཤེས་པ་གཅོད་པའི་རལ་གྲི་དང་ཐོད་པ་ཁྲག་གིས་བཀང་བ་བསྣམས་པ། དབུ་སྐྲ་དམར་སེར་གྱེན་དུ་བརྫེས་པ། མཆེ་བ་གཙིགས་པ་མི་མགོ་རློན་པའི་དོ་ཤལ་དང་སྟག་:གི་པགས་པའི་ཤམ་ཐབས་ཅན་ཅིག་\footnote{གི་ཤམ་ཐབས་ཅན་  སྣར་ཐང་། གི་ཤམ་སྟབས་ཅན་  པེ་ཅིན། }བསམ། དེའི་ཐུགས་ཀར་རཾ་ཡོངས་སུ་གྱུར་པ་ལས་ཉི་མའི་དཀྱིལ་འཁོར་དེའི་སྟེང་ན་ཧཱུཾ་མཐིང་ནག་ལས་གཡས་སུ་ཕཊ་དམར་ནག་ཅིག་ཆད་སོང་བ་ལས་ཉི་མའི་གདན་ལ་དཔལ་ལེགས་ལྡན་ནག་པོ་སྐུ་མདོག་དམར་ནག་ཞལ་གཅིག་ཕྱག་གཉིས་པ། གཡས་གཡོན་ན་ཙནྡན་གྱི་བེ་ཅོན་དང་ཐོད་པ་ཁྲག་གིས་གང་བ་བསྣམས་པ། ཞབས་གཡོན་པ་ཅུང་ཟད་བརྐྱང་བ། གཡས་པ་ཅུང་ཟད་བསྐུམ་པའི་འགྱིང་བག་ཅན། རུས་པའི་ཟེ་ཕྲོག་ཅན་འཇོལ་བེར་སུམ་:བརྩེགས་གཡེལ་\footnote{རྩེགས་གསོལ་  སྣར་ཐང་།  པེ་ཅིན། }བ་གསེར་གྱི་:སྐ་རགས་\footnote{སྐ་རགས་ཅན་  སྣར་ཐང་།  པེ་ཅིན། }བཅིངས་པ་ཅིག་\footnote{ཞིག་  སྣར་ཐང་།  པེ་ཅིན། }ཏུ་བསྒོམ་མོ། །དེ་ནས་:རང་གི་\footnote{རང་གིས་  སྣར་ཐང་།  པེ་ཅིན། }ཐུགས་ཀ་ནས་འོད་འཕྲོས་པས། རི་རབ་ཀྱི་བྱང་ཕྱོགས་ཀྱི་བང་རིམ་ནས་མགོན་པོ་དཔོན་གཡོག་\footnote{གཡོགས་  པེ་ཅིན། }སྤྱན་དྲངས་ལ། ཛཿ་ཧཱུཾ་བཾ་ཧོཿ་ས་བསྟིམ་མོ། །དཔལ་ལེགས་ལྡན་ལ་ཡང་དེ་བཞིན་དུ་བསྟིམ་མོ། །སྐུ་གསུང་ཐུགས་བྱིན་གྱིས་བརླབ་པོ། །ཡང་མགོན་པོའི་གཡོན་ཕྱོགས་སུ་ཡཾ་སྔོ་སྐྱ་ཡོངས་སུ་གྱུར་པ་ལས། །རླུང་གི་དཀྱིལ་འཁོར། དེའི་སྟེང་:ན་བྷྱོཿ་སྨུག་ནག་\footnote{བྷྱོ་རྨུག་གནག་  སྣར་ཐང་།  པེ་ཅིན། }དེ་ལས་འོད་འཕྲོས་སེམས་ཅན་ལ་ཕོག་སྡིག་སྒྲིབ་དག་ཐམས་ཅད་དཔལ་ལྷ་མོའི་སྐུར་སླར་འདུས་པས་དཔལ་ལྷ་མོ་སྐུ་མདོག་སྨུག་ནག་བོང་བུ་ག་དམར་ལ་བཅིབས་པ་:ཞལ་གཅིག་\footnote{ཞལ་ཅིག་  སྣར་ཐང་།  པེ་ཅིན། }ཕྱག་:གཉིས་པ།\footnote{གཉིས་མ།  སྣར་ཐང་།  པེ་ཅིན། } གཡས་གཡོན་ན་རལ་གྲི་དང་ཐོད་པ་ཁྲག་གིས་བཀང་བ་བསྣམས་པ། མཆེ་བ་གཙིགས་པ། དབུ་སྐྲ་དམར་སེར་གྱེན་དུ་བརྫེས་པ། རུས་པའི་རྒྱན་ཆ་ཅན། གོང་སྨེད་གོང་ཐག་སྦྲུལ་ལས་\footnote{ལ་  སྣར་ཐང་།  པེ་ཅིན། }བྱས་པ། སྔ་ཤ་ཕྱི་ཤ་ལ་མི་རོ་བཏགས་\footnote{བཏག་  པེ་ཅིན། }པ། ནད་ཀྱི་:རྐྱལ་པ་\footnote{རྒྱལ་པ་  སྣར་ཐང་། }དང་། མཚོན་གྱི་དྲུ་གུ་:ཐོགས་པ་ཅིག་\footnote{ཐོགས་པ་ཞིག་  སྣར་ཐང་།  པེ་ཅིན། }བསྒོམ་མོ། །དེའི་ཐུགས་ཀ་ནས་རཾ་ཡོངས་སུ་གྱུར་པ་ལས་ཉི་མའི་དཀྱིལ་འཁོར། དེའི་སྟེང་:ན་བྷྱོཿ་སྨུག་ནག་\footnote{དུ་བྷྱོ་སྨུག་གནག་  སྣར་ཐང་།  པེ་ཅིན། }དེ་ལས་གཡོན་ཕྱོགས་སུ་:བྷྱོཿ་ནག་\footnote{ནག་  སྣར་ཐང་།  པེ་ཅིན། }པོ་ཅིག་\footnote{གཅིག་  སྣར་ཐང་།  པེ་ཅིན། }ཆད་:སོང་བ།\footnote{སོང་བས།  སྣར་ཐང་།  པེ་ཅིན། } དཔལ་རེ་མ་ཏི་སྐུ་མདོག་ནག་མོ་རེ་བའི་གོས་གྱོན་པ་དྲེའུ་:ནག་པོ་\footnote{ནག་མོ་  སྣར་ཐང་།  པེ་ཅིན། }ལ་:ཆིབས་པ་\footnote{བཅིབས་པ་  སྣར་ཐང་།  པེ་ཅིན། }ཞལ་གཅིག་ཕྱག་གཉིས་མ་གཡས་གཡོན་ན་རལ་གྲི་དང་ཐོད་པ་ཐོགས་པ། མཆེ་བ་:གཙིགས་པ།\footnote{གཙིགས།  སྣར་ཐང་།  པེ་ཅིན། } སྐྲ་གཟིངས་པ། :གོང་སྨེད་\footnote{གོང་རྨེད་  སྣར་ཐང་།  པེ་ཅིན། }གོང་ཐག་སྦྲུལ་ལས་\footnote{ལ་  སྣར་ཐང་།  པེ་ཅིན། }བྱས་པ། སྔ་ཤ་ཕྱི་ཤ་ལ་མི་རོ་བཏགས་པ། ནད་ཀྱི་རྐྱལ་བུ་དང་མཚོན་གྱི་:དྲུ་གུ་\footnote{དྲུ་བུ་  སྣར་ཐང་། }ཐོགས་པ་ཅིག་\footnote{ཞིག་  སྣར་ཐང་།  པེ་ཅིན། }ཏུ་བསྒོམ་མོ། །དེ་ནས་རང་གི་:ཐུགས་ཀ་\footnote{ཐུགས་  པེ་ཅིན། }ནས་འོད་འཕྲོས་སོང་བས། སུམ་ཅུ་རྩ་གསུམ་གྱི་རྣམ་པར་རྒྱལ་བའི་ཁང་བཟངས་\footnote{གཟངས་  སྣར་ཐང་།  པེ་ཅིན། }ཀྱི་བྱང་ཤར་མཚམས་ཀྱི་མཚོ་མུ་ལེ་ལྡིངས་\footnote{ལྡིང་  སྣར་ཐང་།  པེ་ཅིན། }ནས་ཡེ་ཤེས་ཀྱི་ལྷ་མོ་འཁོར་བཅས་སྤྱན་དྲངས་ལ། ཛཿ་ཧཱུཾ་བཾ་ཧོཿ་ས་བསྟིམ་མོ། །ལྷ་མོ་དྲེལ་ཞོན་མ་ལ་ཡང་དེ་བཞིན་དུ་བསྟིམ་མོ། །དེ་ནས་སྐུ་གསུང་ཐུགས་བྱིན་གྱིས་བརླབ་\footnote{བརློབ་  སྣར་ཐང་།  པེ་ཅིན། }པོ། །དེ་ནས་བཟླས་པ་བྱེད་ན། རང་གི་སྙིང་ག་ནས་རཾ་ཡོངས་སུ་གྱུར་པ་ལས་ཉི་མའི་དཀྱིལ་འཁོར་གྱི་སྟེང་ན་ཧཱུཾ་མཐིང་ནག་བཞེངས་ནས་བཞུགས་པ་ལ། ཨོཾ་ཙཎྜ་མ་ཧཱ་ཀཱ་ལ་བཛྲ་:དཾ་ཥྚི་ན་ཧཱུཾ་\footnote{དཾ་དཾ་ན་ཧཱུཾ་  སྣར་ཐང་། དཾ་དཾ་ན་  པེ་ཅིན། ལྟེ་བ་ནས་ཐོན་རང་གི་ལྟེ་བར་ཡང་རང་གི་ཁ་ནས་ཐོན་མགོན་པོའི་ཞལ་དུ་དེ་བཞིན་དུ་ཁོར་ཡུག་ཏུ་འཁོར་བར་:བསམས་ཤིང་\footnote{བསམ་ཞིང་  སྣར་ཐང་།  པེ་ཅིན། }བཟླས་པར་བྱའོ། །དཔལ་ལྷ་མོའི་:ཟླས་པ་\footnote{བཟླས་པ་  སྣར་ཐང་།
