\chapter{༄༅། །འཇམ་པའི་དབྱངས་དཔའ་བོ་གཅིག་ཏུ་གྲུབ་པའི་སྒྲུབ་ཐབས།}༄༅༅། །རྒྱ་གར་སྐད་དུ། སིད་དྷཻ་\footnote{དྷི་  པེ་ཅིན། }ཀ་བཱི་\footnote{བི་  པེ་ཅིན། }ར་:མཉྫུ་གྷོ་ཥ་སཱ་དྷ་\footnote{མཉྫུ་ཤྲཱིཿསཱ་དྷ་ནཾ་  སྣར་ཐང་། མཉྫུ་ཤྲཱི་སཱ་དྷ་ན་  པེ་ཅིན། }ནཾ། བོད་སྐད་དུ། འཇམ་པའི་དབྱངས་དཔའ་བོ་གཅིག་ཏུ་གྲུབ་པའི་སྒྲུབ་\footnote{སྒྲུབ་པའི་  སྣར་ཐང་། }ཐབས། འཇམ་པའི་དབྱངས་དཔའ་བོ་གཅིག་ཏུ་གྲུབ་པ་ལ་ཕྱག་འཚལ་ལོ། །དེ་ལ་:དང་པོར་\footnote{དང་པོ་  པེ་ཅིན། }སྔགས་པས་གདོང་བཀྲུ་བ་ལ་སོགས་པ་བྱ་སྟེ་ཡིད་དང་མཐུན་པའི་བསམ་གཏན་གྱི་གནས་གཙང་ཞིང་། མེ་ཏོག་དྲི་ཞིམ་པོས་གཅལ་དུ་བཀྲམ་པར། སྟན་གཙང་ལ་:བདེ་བར་སྐྱིལ་\footnote{བདེ་བརདཀྱིལ་  པེ་ཅིན། }མོ་ཀྲུང་\footnote{དཀྲུང་  པེ་ཅིན། }གིས་འདུག་སྟེ། འབྱུང་པོ་ཐམས་ཅད་པའི་གཏོར་མའི་སྔགས་ཀྱིས་གཏོར་མ་བྱིན་གྱིས་བརླབས་ལ། ཨོཾ་:ཨཱཿ་བིགྷྣཱནྟ་ཀྲྀཏ་\footnote{ཨ་བི་གྷྣཱཾ་ཏ་ཀྲྀཏ་  སྣར་ཐང་། ཨ་བིགྷ་ནཱན་ཏ་ཀྲིད་  པེ་ཅིན། }ཧཱུཾ་ཕཊ་ཅེས་བྱ་བའི་སྔགས་ལན་བཅུའི་བར་དུ་བརྗོད་དེ་ཕྱོགས་བཅུ་ན་གནས་པའི་བགེགས་ཐམས་ཅད་བསྐྲད་པར་བྱའོ། །དེ་ནས་རང་གི་སྙིང་གར་ཨ་ཆོས་ཐམས་ཅད་ཆོས་ཀྱི་དབྱིངས་ཀྱི་རང་བཞིན་སྐྱེ་བ་:མེད་པར་\footnote{མེད་པ་  སྣར་ཐང་།  པེ་ཅིན། }སྟོན་པར་བྱེད་པ། སྟོན་ཀའི་ཟླ་བ་ལྟར་དཀར་བ། འོད་ཟེར་དཀར་པོ་:འཕྲོས་པས་\footnote{འཕྲོ་བས་  སྣར་ཐང་།  པེ་ཅིན། }རང་གི་ལུས་གསལ་བར་བྱེད་པ་ཤིན་ཏུ་ཕྲ་བ་བསམས་ལ། དེ་ཡོངས་སུ་གྱུར་པས་འོད་ཟེར་དཀར་པོའི་ཐིག་ལེ་ཤེལ་སྒོང་དག་པ་ལྟ་བུ། དེ་རིམ་གྱིས་འཕེལ་བར་གྱུར་པས་ཟླ་བའི་དཀྱིལ་འཁོར་གྱི་གཟུགས་སུ་བསམ་ཞིང་། དེའི་སྟེང་དུ་ཡི་གེ་དྷཱིཿ་\footnote{དྷི་་  སྣར་ཐང་།  པེ་ཅིན། }དེའི་འོད་ཟེར་གྱི་ཚོགས་འཕྲོ་བ་རིན་པོ་ཆེའི་སྒྲོན་མ་ལྟར་དཀར་བ་རྣམ་པར་བསམ་མོ། །དེ་ནས་ཡི་གེ་དྷཱིཿ་\footnote{དྷི་་  པེ་ཅིན། }དེའི་འོད་ཟེར་ཤིན་ཏུ་དཀར་བའི་:ཚོགས་ཀྱིས་\footnote{ཚོགས་ཀྱི་  པེ་ཅིན། }འཇིག་རྟེན་གྱི་ཁམས་ཐམས་ཅད་གསལ་བར་བྱས་པ་སྔོན་དུ་འགྲོ་བས་འཕགས་པ་འཇམ་དཔལ་ལ་སོགས་པ་སངས་རྒྱས་དང་བྱང་ཆུབ་སེམས་དཔའ་དང་། བླ་མ་རྣམས་བསྐུལ་ཏེ་སྤྱན་དྲངས་ལ་མདུན་གྱི་ནམ་མཁའི་ཕྱོགས་ལ་རྣམ་པར་བསམས་ལ་ཕྱག་བྱའོ། །དེའི་འོག་ཏུ་:ཡིད་ཀྱིས་\footnote{ཡིད་ཀྱི་  སྣར་ཐང་།  པེ་ཅིན། }རང་བཞིན་གྱི་མཆོད་པས་མཆོད་པར་བྱ་བ་ཡིན་ལ། དེ་ལ་མཆོད་པའི་ཆོ་གའི་རིམ་པ་ནི་འདི་ཡིན་ཏེ། ཨོཾ་བཛྲ་པུཥྤེ་ཧཱུཾ་ཞེས་བྱ་བའི་སྔགས་བརྗོད་ལ། ཡི་གེ་དྷཱིཿ་\footnote{དྷིཿ་  སྣར་ཐང་། དྷི་  པེ་ཅིན། }ལས་བྱུང་བའི་མེ་ཏོག་གི་མཆོད་པའི་ཚོགས་ཀྱིས་མཆོད་པར་བྱའོ། །དེ་ནས་ཨོཾ་བཛྲ་དྷཱུ་\footnote{དྷཱུཾ་  སྣར་ཐང་། དྷུ་  པེ་ཅིན། }པེ་ཧཱུཾ་ཞེས་བྱ་བའི་སྔགས་བརྗོད་ལ། ཡི་གེ་:དྷཱིཿ་དེ་ཉིད་\footnote{དྷི་དེ་ཉིད་ལས་  པེ་ཅིན། }ལས་ཡང་དག་པར་སྤྲོས་པའི་བདུག་པའི་མཆོད་པའི་ཚོགས་ཀྱིས་མཆོད་པར་བྱའོ། །དེའི་འོག་ཏུ་ཨོཾ་བཛྲ་དཱི་པེ་ཧཱུཾ་ཞེས་བྱ་བའི་སྔགས་བརྗོད་ལ། ཡི་གེ་:དྷཱིཿ་དེ་\footnote{དྷི་དེ་  པེ་ཅིན། }ཉིད་ལས་རིན་པོ་ཆེའི་སྒྲོན་མ་སྤྲོས་ལ་མཆོད་པར་བྱའོ། །དེ་ནས་ཨོཾ་བཛྲ་གནྡྷེ་ཧཱུཾ་ཞེས་བྱ་བའི་སྔགས་བརྗོད་ལ། ཡི་གེ་དྷིཿ་\footnote{དྷི་  པེ་ཅིན། }དེ་ཉིད་ལས་བྱུང་བའི་དྲིའི་མཆོད་པའི་སྤྲིན་ཡང་དག་པར་སྤྲོས་པས་མཆོད་པར་བྱའོ། །དེ་ནས་ཨོཾ་བཛྲ་ནཻ་བི་དྱེ་\footnote{དྱཻ་  སྣར་ཐང་། ཏྱེ་  པེ་ཅིན། }ཧཱུཾ་ཞེས་བྱ་བའི་སྔགས་བརྗོད་པས།\footnote{ལ།  པེ་ཅིན། } ཡི་གེ་:དྷཱིཿ་དེ་\footnote{དྷི་དེ་  པེ་ཅིན། }ཉིད་ལས་རིན་པོ་ཆེ་སྣ་ཚོགས་ཀྱི་སྣོད་དུ་བཅུག་པའི་ཞལ་ཟས་རོ་མཆོག་དང་:ལྡན་པའི་མཆོད་པའི་\footnote{ལྡན་པའི་  པེ་ཅིན། }སྤྲིན་རྣམ་པ་སྣ་ཚོགས་ཡང་དག་པར་སྤྲོས་པས་མཆོད་པར་བྱའོ། །དེ་ནས་ཨོཾ་སརྦ་\footnote{བཛྲ་  པེ་ཅིན། }བི་ཤིཥྚ་\footnote{ཤིཤྛ་  པེ་ཅིན། }པཱུ་ཛ་མེ་གྷ་པྲ་ས་ར་ས་མུ་:དྲ་ཨཱཿ་\footnote{ཏྲ་ཨ་་  པེ་ཅིན། }ཧཱུཾ་ཞེས་བྱ་བའི་སྔགས་བརྗོད་ལ། དེ་ཉིད་ལས་རིན་པོ་ཆེ་སྣ་ཚོགས་ཀྱི་གདུགས་དང་། རྒྱལ་མཚན་དང་། དྲིལ་བུ་དང་། བ་དན་དང་། བླ་རེ་ལ་སོགས་པ་དང་། འཁོར་ལོས་:སྒྱུར་བའི་\footnote{བསྒྱུར་བའི་  པེ་ཅིན། }རིན་པོ་ཆེ་སྣ་བདུན་གྱིས་མཆོད་པར་བྱའོ། །དེ་ལྟར་མཆོད་པ་བྱས་ནས་དེ་དག་གི་སྤྱན་སྔར་སྡིག་པ་བཤགས་པ་དང་། :བསོད་ནམས་ལ་\footnote{བསོད་ནམས་ཀྱི་  སྣར་ཐང་།  པེ་ཅིན། }རྗེས་སུ་ཡི་རང་བ་དང་གསོལ་བ་གདབ་པ་དང་། བསྐུལ་བ་དང་། བསོད་ནམས་ཡོངས་སུ་བསྔོ་བ་དང་། གསུམ་ལ་སྐྱབས་སུ་འགྲོ་བ་བྱས་ལ། བྱམས་པ་དང་། སྙིང་རྗེ་དང་། དགའ་བ་དང་། བཏང་སྙོམས་ཏེ་ཚངས་པའི་གནས་བཞི་བསྒོམས་\footnote{སྒོམས་  སྣར་ཐང་། }ནས་བྱང་ཆུབ་ཀྱི་སེམས་བརྟན་པར་བྱའོ། །དེ་ནས་ཨོཾ་སྭ་:བྷཱ་བ་བི་ཤུདྡྷཿ་སརྦ་དྷརྨཱཿ་སྭ་བྷཱ་ཝ་བི་ཤུདྡྷོ྅\footnote{བྷཱ་བི་ཤུ་དདྷཿསརྦ་དྷརྨཱཿསྭ་བྷཱ་བ་ཤུདྡྷོ་  སྣར་ཐང་། བྷ་བ་ཤུདྷཿསརྦ་དྷརྨྨཱ་སྭཱ་བྷཱ་བ་ཤུདྡྷོ་  པེ་ཅིན། }ཧཾ་ཞེས་བྱ་བའི་སྔགས་ལན་བདུན་བརྗོད་ལ། འགྲོ་བ་ཐམས་ཅད་སྒྱུ་མ་དང་། རྨི་ལམ་དང་:སྨིག་རྒྱུ་\footnote{མིག་སྒྱུ་  པེ་ཅིན། }ལྟ་བུར་སྟོང་པ་ཉིད་ཀྱི་རང་བཞིན་དུ་དམིགས་པར་བྱའོ། །དེ་ནས་རང་གི་སྙིང་གར་ཆུད་པའི་ཡི་གེ་དྷཱིཿ་\footnote{དྷི་་  པེ་ཅིན། }དེའི་འོད་ཟེར་གྱི་ཚོགས་སྤྲོས་པས་རང་གི་ལུས་སྟོང་པར་བལྟ་ཞིང་། དེ་ནས་བདག་ཉིད་འཇམ་དཔལ་ཡེ་ཤེས་ཀྱི་སྐུའི་བདག་ཉིད་ཅན་ཡི་གེ་དྷཱིཿ་\footnote{དྷི་་  པེ་ཅིན། }ཙམ་དུ་གནས་པར་བྱའོ། །ཕྱིས་ཡི་གེ་དྷཱེཿ་\footnote{དྷི་  པེ་ཅིན། }དེ་ཡོངས་སུ་གྱུར་པའི་ཨུཏྤལ་སྔོན་པོ་:བསྒོམས་པ་\footnote{བསྒོམས་  པེ་ཅིན། }ལ་དེའི་དབུས་:སུ་ཡང་\footnote{སུ་  པེ་ཅིན། }ཡི་གེ་དྷཱིཿ་\footnote{དྷི་  པེ་ཅིན། }གནས་པར་བསམ་མོ། །འོད་ཟེར་དེ་ཡིས་འགྲོ་བ་ཀུན། །ཀུན་ནས་ཡོངས་སུ་སྦྱངས་ནས་ནི། །རང་གི་འདོད་ལྷའི་གཟུགས་དག་ཏུ། །གྱུར་པ་ཨུཏྤལ་ལ་\footnote{ལར་  སྣར་ཐང་། }བསྟིམ་མོ། །དེ་ནས་དེ་ཡོངས་སུ་གྱུར་པ་ལས་ཤེལ་སྒོང་དག་པ་དང་འདྲ་བ། དབུ་གཅིག་པ་དང་ཕྱག་གཉིས་པ། །ཟླ་བའི་གདན་བཞུགས་ཟླ་བ་འདྲའོ། །ཟུར་ཕུད་ལྔ་པོ་དག་གི་ཐོད། །ཞི་བ་རིན་ཆེན་ཅོད་པན་འཆང་། །རྒྱན་ཀུན་རྣམ་པར་འཛིན་འགྱུར་དང་། །གཡས་:ཀྱིས་མཆོག་\footnote{ཀྱིས་མཆོད་  སྣར་ཐང་། ཀྱི་མཆོད་  པེ་ཅིན། }སྦྱིན་གཡོན་:དག་གིས།\footnote{དག་གི།  པེ་ཅིན། } །ཨུཏྤལ་སྔོན་པོ་འཛིན་འགྱུར་པ། །ཨུཏྤལའི་སྟེང་དུ་ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པའི་གླེགས་བམ་ཡང་དག་པར་འཛིན་པའོ། །འཇམ་པའི་དབྱངས་སུ་བདག་ཉིད་བསྒོམས་ལ། རང་གི་ལུས་ལས་འོད་ཟེར་འཕྲོས་པས་ནམ་མཁའི་ཁམས་མ་:ལུས་པ་\footnote{ལུས་པར་  པེ་ཅིན། }ཁྱབ་པར་བྱའོ། །དེའི་འོག་ཏུ་ཉིད་ཀྱི་ཐུགས་ཀའི་ཟླ་བའི་དཀྱིལ་འཁོར་ལ་ཡི་གེ་དྷཱིཿ་\footnote{དྷི་  པེ་ཅིན། }དཀར་པོ་འབར་བའི་ཕྲེང་བ་:གཏམས་པར་བསམས་\footnote{གཏིབས་པར་བསམ་  སྣར་ཐང་། གཏིབས་པ་བསམས་  པེ་ཅིན། }ལ། དེ་ནས་རང་གི་སྙིང་ག་ལ་གནས་པའི་ཡི་གེ་དྷཱིཿ་\footnote{དྷི་  པེ་ཅིན། }ལས་འོད་ཟེར་གྱི་ཚོགས་ཀྱི་སྤྲིན་ཀུན་ཏུ་སྤྲོས་པ་དེས། དམ་ཚིག་སེམས་དཔའ་དང་འདྲ་བའི་ཡེ་ཤེས་སེམས་དཔའ་བསྐུལ་ནས། སྤྱན་དྲངས་པ་མདུན་གྱི་ནམ་མཁའ་ལ་རྣམ་པར་བསམས་ལ། སྔ་མ་བཞིན་ཡང་དག་པར་མཆོད་དེ། ཛཿ་ཧཱུཾ་བཾ་ཧོཿ་ཞེས་བྱ་བའི་ཡི་གེ་:བཞིས་སྤྱན་\footnote{བཞིས་  སྣར་ཐང་།  པེ་ཅིན། }དྲངས་ནས། དམ་ཚིག་སེམས་དཔའ་བདག་ཉིད་ལ་དགུག་པ་དང་:། གཞུག་པ་དང་། \footnote{།    སྣར་ཐང་། }བཅིང་བ་དང་། དགྱེས་པ་བྱས་ལ། ཨོཾ་དྷརྨ་\footnote{དྷརྨཱ་  སྣར་ཐང་། དྷརྨྨ་  པེ་ཅིན། }དྷཱ་:ཏུ་བཱ་གཱི་\footnote{ཏུ་བ་གི་  པེ་ཅིན། }ཤྭ་རི་\footnote{ར་  པེ་ཅིན། }སྭ་བྷཱ་བ་:ཨཱཏྨ་ཀོ྅ཧཾ།\footnote{ཨཏྨ་ཀོཉཧཾ  ཅོ་ནེ། ཨཱད་མ་ཀོ་ཧཾ།  པེ་ཅིན། } ཞེས་ང་རྒྱལ་བྱའོ། །དེའི་རྗེས་ལ་ཡི་གེ་དྷཱིཿ་\footnote{དྷིཿ་  པེ་ཅིན། }ལས་འོད་ཟེར་སྤྲོ་བ་བྱས་ལ་དེ་བཞིན་གཤེགས་པ་མདུན་གྱི་ནམ་མཁའ་ལ་རྣམ་པར་བསམས་ལ། དབང་བསྐུར་བའི་ཚིགས་སུ་བཅད་པ་འདིས་དོན་དུ་:མཉེས་པར་\footnote{གཉེར་བར་  པེ་ཅིན། }བྱ་སྟེ། བྱང་ཆུབ་རྡོ་རྗེ་སངས་རྒྱས་ལ། །ཇི་ལྟར་མཆོད་ཆེན་སྩོལ་\footnote{གསོལ་  སྣར་ཐང་།  པེ་ཅིན། }མཛད་པ། །བདག་ལའང་བསྲུང་བའི་དོན་གྱི་ཕྱིར། །:ནམ་མཁའི་\footnote{ནམ་མཁའ་  སྣར་ཐང་། }རྡོ་རྗེ་དེང་\footnote{དང་  སྣར་ཐང་། }བདག་སྩོལ།\footnote{གསོལ།  སྣར་ཐང་།  པེ་ཅིན། } །དེ་ནས་དེ་བཞིན་གཤེགས་པ་དེ་དག་གི་སྐུ་ལས་བྱུང་བའི་ལོ་ཙ་\footnote{ཙཱ་  ཅོ་ནེ། }ན་དང་། མཱ་མ་ཀཱི་དང་། གོས་དཀར་མོ་དང་། :ཏཱ་རས་\footnote{ཏཱ་ར་  སྣར་ཐང་།  པེ་ཅིན། }རིན་པོ་ཆེའི་བུམ་པ་ཡེ་ཤེས་ཀྱི་བདུད་རྩིས་བཀང་བ་ཕྱག་ན་བསྣམས་པས། རྡོ་རྗེ་ཆེན་པོའི་དབང་བསྐུར་བ། །ཁམས་གསུམ་པོ་ཡིས་ཕྱག་བྱས་པ། །གསང་བ་གསུམ་གྱི་གནས་ལས་བྱུང་། །སངས་རྒྱས་ཀུན་གྱིས་སྦྱིན་པར་བྱ། །ཞེས་བྱ་བ་འདོན་ཅིང་དབང་བསྐུར་བར་བསམ་མོ། །དབང་:བསྐུར་བ་\footnote{བསྐུར་  སྣར་ཐང་། }རྫོགས་མ་ཐག་རིན་པོ་ཆེའི་ཅོད་པན་གྱི་སྟེང་དུ། དེ་བཞིན་གཤེགས་པ་མི་:བསྐྱོད་པ་\footnote{སྐྱོད་པ་  པེ་ཅིན། }བསམས་ལ། ལུས་དང་ངག་དང་ཡིད་གསུམ་:བྱིན་གྱིས་\footnote{བྱིན་གྱི་  པེ་ཅིན། }བརླབ་པའི་ཕྱིར། སྤྱི་བོ་ལ་:ཡི་གེ་\footnote{ཡེ་ཤེག་  སྣར་ཐང་། }ཨོཾ་དཀར་པོ་ཟླ་བའི་དཀྱིལ་འཁོར་ལ་གནས་པ་དང་། ལྐོག་མར་ཡི་གེ་ཨཱཿ་\footnote{ཨ་  པེ་ཅིན། }དམར་པོ་ཟླ་བའི་དཀྱིལ་འཁོར་ལ་གནས་པ་དང་། སྙིང་གར་ཡི་གེ་ཧཱུཾ་ནག་པོ་ཟླ་བའི་དཀྱིལ་འཁོར་ལ་གནས་པ་རྣམ་པར་བཀོད་པ་ལ། སྤྲོ་བ་དང་བསྡུ་\footnote{སྡུ་  པེ་ཅིན། །སྔགས་ཇི་ཙམ་:འོས་བ་\footnote{འདོད་བ་  སྣར་ཐང་།  པེ་ཅིན། }བཟླས་པ་བྱས་ནས། ཆོ་ག་ཉམས་པ་བསྐང་བར་བྱ་བའི་ཕྱིར་རྡོ་རྗེ་སེམས་དཔའི་སྙིང་པོ་ཡི་གེ་བརྒྱ་པ་ཞེས་བྱ་བའི་སྔགས་བཟླས་ནས། ཡེ་ཤེས་སེམས་དཔའ་ཡང་དག་པར་མཆོད་དེ་བསོད་ནམས་སུ་བསྔོས་\footnote{སྔོས་  པེ་ཅིན། }ལ་གཤེགས་སུ་གསོལ་ལོ། །དེ་ལ་ཡེ་ཤེས་སེམས་དཔའ་:གཤེགས་སུ་\footnote{གཤེགས་  སྣར་ཐང་། }གསོལ་བའི་ཚིགས་སུ་བཅད་པ་ནི།
