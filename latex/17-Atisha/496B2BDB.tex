\chapter{༄༅། །འཕགས་མ་སྒྲོལ་མའི་སྒྲུབ་ཐབས།}༄༅༅། །རྒྱ་གར་སྐད་དུ། ཨཱརྱ་:ཏཱ་རཱ་\footnote{ཏཱ་ར་  སྣར་ཐང་།  པེ་ཅིན། }སཱ་དྷ་ནཾ། བོད་སྐད་དུ། འཕགས་མ་སྒྲོལ་མའི་སྒྲུབ་ཐབས། འཕགས་མ་སྒྲོལ་མ་ལ་ཕྱག་འཚལ་ལོ། །འཇིག་རྟེན་གསུམ་མགོན་ཡུམ་གྱུར་སྤྱན་ཡངས་མ། །སྒྲོལ་མའི་ཞབས་ཀྱི་པདྨོར་\footnote{པདྨར་  སྣར་ཐང་།  པེ་ཅིན། }རབ་བཏུད་ནས། །ཡུན་རིངས་བླ་མ་བསྟེན་ལས་གང་འོངས་པའི། །སྒྲུབ་ཐབས་ཆོ་ག་བསྡུས་པ་བཤད་པར་བྱ། །ཡིད་འོང་གནས་:ནི་གང་ཡང་རུང་།\footnote{ནི་ལ་ལ་རུ།  སྣར་ཐང་།  པེ་ཅིན། } །མེ་ཏོག་དྲི་ཞིམ་གཅལ་:བཀྲམ་སྟེ།\footnote{བཀྲམ་ཏེ།  སྣར་ཐང་།  པེ་ཅིན། } །:སྤོས་ཀྱིས་\footnote{སྤོས་ཀྱི་  སྣར་ཐང་།  པེ་ཅིན། }ཉམས་དགར་བྱས་ནས་ནི། །དེ་དབུས་སྟན་བཟང་བདེ་བ་ལ། །སྲིད་པའི་བདེ་བ་ཡོངས་སྤངས་པའི། །རྣལ་འབྱོར་:ཅན་གྱིས་\footnote{ཅན་གྱི་  སྣར་ཐང་།  པེ་ཅིན། }ཉེར་འདུག་ནས། །སྙིང་གའི་ས་བོན་ལས་བྱུང་བའི། །འོད་ཟེར་:ཚོགས་ཀྱིས་\footnote{ཚོགས་ཀྱི་  སྣར་ཐང་།  པེ་ཅིན། }བླ་མ་དང་། །སངས་རྒྱས་བྱང་ཆུབ་སེམས་དཔའ་རྣམས། །སྤྱན་དྲངས་:དེ་བདག་\footnote{དེ་དག་  སྣར་ཐང་།  པེ་ཅིན། }མདུན་ཉིད་དུ། །མཆོད་པས་ཡང་དག་རབ་མཆོད་ནས། །དཀོན་མཆོག་གསུམ་སོགས་རྗེས་དྲན་བྱ། །ཨོཾ་སྭ་:བྷཱ་བ་ཤུདྡྷཿ་སརྦ་དྷརྨཱཿ་སྭ་བྷཱ་བ་ཤུདྡྷོ྅\footnote{བྷཱ་བ་ཤུདྡྷསརྦ་དྷརྨ་སྭ་བྷ་བ་ཤུདྡྷོ་  སྣར་ཐང་། བྷཱ་བ་ཤུདྡྷསརྦ་དྷརྨ་སྭཱ་བྷ་ཤུདྡྷོ་  པེ་ཅིན། }ཧཾ། ཨོཾ་ཤཱུ་ནྱ་ཏཱ་ཛྙཱ་ན་བཛྲ་སྭ་བྷཱ་བ་:ཨཱཏྨ་ཀོ྅\footnote{ཨཱཏྨ་  སྣར་ཐང་།  པེ་ཅིན། }ཧཾ། དེ་ནས་བརྟན་པ་དང་གཡོ་བ་ཐམས་ཅད་སྟོང་པ་ཉིད་དུ་རྣམ་པར་བསམ་མོ། །དེ་ནས་པཾ་ལས་བྱུང་བའི་:པདྨ་ཚད་\footnote{པདྨ་  སྣར་ཐང་།  པེ་ཅིན། }ཆེན་པོའི་དབུས་སུ་རབ་ཏུ་རྫོགས་པའི་ཟླ་བའི་དཀྱིལ་འཁོར་ལ་ཡི་གེ་ཏཱཾ་ལྗང་གུ་བསམས་ཏེ། དེ་ཡོངས་སུ་གྱུར་པ་ལས་:བདག་ཉིད་ཡི་དམ་སྒྲོལ་མ་\footnote{བདག་ཉིད་  སྣར་ཐང་།  པེ་ཅིན། }སྐུ་མདོག་ལྗང་གུ་ཞལ་གཅིག་ཕྱག་གཉིས་པ། གཡས་པ་མཆོག་སྦྱིན་པ། གཡོན་པ་ཨུཏྤལ་འཛིན་པ། རྒྱན་ཐམས་ཅད་ཀྱིས་བརྒྱན་པ། ལོ་བཅུ་དྲུག་ལོན་པ་ལྟ་བུ། གཟི་བརྗིད་བཟོད་པར་དཀའ་བ་དང་ལྡན་པ། ཞལ་ཅུང་ཟད་འཛུམ་པ་དང་ལྡན་པ། སྐྲའི་ཚོགས་ལྷག་པར་བཅིངས་ཤིང་ཟླ་བའི་འོད་ཅན། རྡོ་རྗེ་སྐྱིལ་མོ་ཀྲུང་གིས་ཟླ་བའི་དཀྱིལ་འཁོར་མནན་པ། སྤྱན་པདྨ་གཞོན་ནུའི་འདབ་མ་ལྟ་བུ། ཅུང་ཟད་འཛུམ་པ་མདུན་ཉིད་དུ་གཟིགས་པའི་འཕགས་མ་སྒྲོལ་མ་ནུ་མའི་དཀྱིལ་འཁོར་:མཁྲེགས་པ་\footnote{མཁྲེང་བ་  སྣར་ཐང་།  པེ་ཅིན། }དང་ལྡན་པ་རྣམ་པར་བསམ་མོ། །དེ་ནས་རང་གི་སྙིང་གའི་ཏཱཾ་ལས་འོད་ཟེར་བྱུང་བ་རྣམས་:ཀྱིས་རྗེ་བཙུན་མ་ཡང་དག་པར་བསྐུལ་ལ། ས་བོན་ལས་བྱུང་བའི་མཆོད་པ་རྣམས་ཀྱིས་\footnote{ཀྱིས་ཡེ་ཤེས་སེམས་དཔའ་བཀུག་སྟེ་  སྣར་ཐང་།  པེ་ཅིན། }ཡང་དག་པར་མཆོད་ནས་ཨུཏྤ་ལའི་ཕྱག་རྒྱས་རང་གི་ལུས་ལ་བསྟིམ་པར་བྱའོ། །དེ་ནས་གནས་ཇི་ལྟ་བ་རྣམས་སུ་རྣམ་པར་སྣང་མཛད་ལ་སོགས་པ་རྣམས་བསམ་ཞིང་། དེ་བཞིན་གཤེགས་པ་རྣམས་དབང་བསྐུར་བ་དོན་དུ་གཉེར་ཏེ། བདག་གི་དབུ་རྒྱན་ལ་གསུང་རྡོ་རྗེས་རྒྱས་གདབ་པར་བྱའོ། །དེའི་འོག་ཏུ་རང་གི་སྙིང་གར་ཟླ་བའི་དཀྱིལ་འཁོར་ལ་ཡི་གེ་ཏཱཾ་ལ་སོགས་པ་རྣམ་པར་:བཀོད་པར་བྱ།\footnote{བཀོད་དེ།  སྣར་ཐང་།  པེ་ཅིན། } སྤྲོ་བ་དང་བསྡུ་པའི་རིམ་པས་བསྒོམ་པར་བྱའོ། །གང་གི་ཚེ་སྐྱོ་བ་ན། ཨོཾ་ཏཱ་རེ་ཏུཏྟཱ་རེ་ཏུ་རེ་སྭཱ་ཧཱ། ཞེས་བྱ་བའི་སྔགས་བཟླས་སོ། །དེ་ནས་གང་གི་ཚེ་ལྡང་བར་འདོད་པ་ན་རང་གི་སྙིང་གར་གནས་པའི་ས་བོན་ལས་འོད་ཟེར་འཕྲོས་ཏེ། དེས་སེམས་ཅན་:ཐམས་ཅད་\footnote{ཐམས་ཅད་དྲངས་  སྣར་ཐང་། ཐམས་ཅད་དྲང་  པེ་ཅིན། }:ལ་ཕོག་\footnote{ལ་  སྣར་ཐང་།  པེ་ཅིན། }སླར་ཡང་འོད་ཟེར་སྤྲོས་པས་:དེ་དག་གི་\footnote{དེ་དག་  སྣར་ཐང་།  པེ་ཅིན། }སྙིང་གར་ཞུགས་པར་བལྟ་\footnote{ལྟ་  སྣར་ཐང་།  པེ་ཅིན། }ཞིང་། དེ་ཉིད་ཀྱིས་དེ་དག་ཐམས་ཅད་ལྷ་མོའི་སྐུར་གྱུར་པར་ལྷག་པར་མོས་པར་བྱའོ། །དེ་ནས་སླར་ཡང་ས་བོན་ལས་བྱུང་བའི་འོད་ཟེར་བདུད་རྩིའི་རང་བཞིན་:རྣམས་ཀྱི་\footnote{རྣམས་ཀྱིས་  སྣར་ཐང་།  པེ་ཅིན། }གཏོར་མའི་སྣོད་ཡོངས་སུ་བཀང་ལ། ཨོཾ་ཨ་ཀཱ་རོ་མུ་ཁཾ་སརྦ་དྷརྨཱ་ཎཱཾ་ཨཱདྱ་\footnote{ཨ་དྱ་  སྣར་ཐང་།  པེ་ཅིན། }ནུཏྤནྣ་ཏྭཱཏ་ཨོཾ་ཨཱཿ་ཧཱུཾ་ཕཊ་སྭཱ་ཧཱ། ཞེས་བྱ་བས་བྱིན་གྱིས་བརླབས་ལ་ཕུལ་ལ། དེ་ནས་ཡེ་ཤེས་སེམས་དཔའ་མདུན་དུ་བཞུགས་པ་ལ་དམིགས་པས་ཡི་གེ་བརྒྱ་པ་ཡོངས་སུ་བཟླས་ནས། ལེགས་པར་མཆོད་དེ། མངོན་པར་འདོད་པའི་དངོས་གྲུབ་:གསོལ་བ་\footnote{གསོལ་  སྣར་ཐང་།  པེ་ཅིན། }བཏབ་ལ། :ཨོཾ་ཏཱ་\footnote{ཨོཾ་ཏུ་  སྣར་ཐང་། }རེ་ཏུཏྟཱ་རེ་ཏུ་རེ་མུཿ་ཞེས་པས་གཤེགས་སུ་:གསོལ་ལོ།\footnote{གསོལ།  སྣར་ཐང་། } །རིམ་པ་འདིས་ཐུན་མཚམས་བཞིར་:བསྒོམས་ན་\footnote{བསྒོམས་ནས་  སྣར་ཐང་།  པེ་ཅིན། }མྱུར་:བར་དངོས་གྲུབ་ཐོབ་\footnote{བར་འགྲུབ་  སྣར་ཐང་།  པེ་ཅིན། }པར་འགྱུར་རོ། །ད་ནི་:བསྲུང་བའི་\footnote{སྲུང་བའི་  སྣར་ཐང་།  པེ་ཅིན། }སྦྱོར་བ་ཅུང་ཟད་བཤད་པར་བྱ་སྟེ། སྔར་བཤད་པའི་ཚུལ་གྱིས་རྩ་བའི་སྔགས་འབུམ་ཕྲག་བཅུ་བཟླས་ཤིང་སྒྲོལ་མའི་ཏིང་ངེ་འཛིན་ལ་གནས་པའི་རྣལ་འབྱོར་པས་བཟླས་པའི་ཆོ་ག་སྔ་མ་ལྟ་བུ་ཉིད་ལས་ཁྱད་པར་དུ་བསྒྲུབ་བྱའི་མིང་སྔགས་ཀྱི་དབུས་ན་གནས་པར་བསམ་ལ་:བཟླས་པ་\footnote{ཟླས་པ་  སྣར་ཐང་། }ཇི་སྲིད་འདོད་ཀྱི་བར་དུ་བྱའོ། །དེ་ནས་ས་བོན་གྱི་ཡི་གེ་ལས་འོད་ཟེར་བྱུང་བས་བསྒྲུབ་བྱའི་ལུས་ཐམས་ཅད་ཁྱབ་པར་བལྟས་ཏེ། དེས་ལུས་རབ་ཏུ་ཚིམ་པར་གྱུར་ཅིང་ཉེ་བར་འཚེ་བ་ཐམས་ཅད་དང་བྲལ་བར་མོས་པར་བྱའོ། །འདི་ནི་ཞི་བའི་སྲུང་\footnote{བསྲུང་  སྣར་ཐང་།  པེ་ཅིན། }བའོ། །དེ་ནས་ཕྱིར་བཟློག་པའི་:སྲུང་བ་\footnote{བསྲུང་བ་  སྣར་ཐང་།  པེ་ཅིན། }བཤད་པ། ཆོ་ག་སྔ་མ་དང་ལྡན་པའི་རྣལ་འབྱོར་པས་རང་གི་སྙིང་གར་ཏཱཾ་ལྗང་གུ་འོད་ཟེར་གྱི་ཕྲེང་བ་འཁྲུགས་པར་བསམས་ལ། དེ་ལས་བྱུང་བའི་འོད་ཟེར་གྱི་ཚོགས་འབར་བས་གདུག་པའི་ལུས་གཅེར་བུ་སྐྲ་:གྲོལ་བ་\footnote{གྲོལ་བར་  སྣར་ཐང་།  པེ་ཅིན། }འདར་ཞིང་སྐྱབས་དང་བྲལ་བའི་སྙིང་གར་ཞུགས་པས་དེ་རེངས་པ་དང་མནན་པ་དང་། རྨོངས་པ་དང་། བཅིངས་པར་རྣམ་པར་བསམ་མོ། །གང་གི་ཚེ་དེ་ལྟ་བུའི་ཏིང་ངེ་འཛིན་གྱིས་:སྐྱོ་བ་\footnote{སྐྱོ་  སྣར་ཐང་།  པེ་ཅིན། }ན་ས་བོན་གྱི་ཡི་གེ་ལ་དམིགས་པས་གསང་སྔགས་བཟླས་པ་བྱ་སྟེ། ན་:མཿ་ཨཱརྱཱ་\footnote{ཨཱརྱཱ་  སྣར་ཐང་།  པེ་ཅིན། }བ་ལོ་ཀི་ཏེ་ཤྭ་རཱ་ཡ། བོ་དྷི་ས་ཏྭཱ་ཡ། མ་ཧཱ་ས་ཏྭཱ་ཡ། མ་ཧཱ་ཀཱ་རུ་ཎི་ཀཱ་ཡ། ཏདྱ་ཐཱ། ཨོཾ་ཏཱ་རེ་ཏུཏྟཱ་རེ་:ཏུ་རེ་\footnote{ཏུ་རེ་སྭཱ་ཧཱ་  སྣར་ཐང་།  པེ་ཅིན། }སརྦ་དུཥྚཱན།\footnote{དུཥྚན།  སྣར་ཐང་།  པེ་ཅིན། } པྲ་དུཥྚཱ་ན།\footnote{དུཥྚན།  སྣར་ཐང་།  པེ་ཅིན། } མ་མ་ཀྲྀ་ཏེ་ཛཾ་བྷ་ཡ། སྟཾ་བྷ་ཡ། :མོ་ཧ་\footnote{མ་ཧཱ་  སྣར་ཐང་།  པེ་ཅིན། }ཡ། བནྡྷ་ཡ། ཧཱུཾ་ཧཱུཾ་ཧཱུཾ། ཕཊ་ཕཊ་ཕཊ། :སརྦྦ་དུཥྚཱན་\footnote{སརྦ་དུཥྚན་  སྣར་ཐང་།  པེ་ཅིན། }སྟཾ་བྷ་ནི་ཏཱ་རེ་སྭཱ་ཧཱ། སྲུང་བ་འདི་དག་ཀྱང་ཐུན་མཚམས་བཞིར་བསྒོམ་པར་བྱའོ། །དེ་བཞིན་དུ་ལམ་དུ་འགྲོ་བས་ཏིང་ངེ་འཛིན་དང་བཅས་པས་མདུད་པ་དོར་\footnote{བོར་  སྣར་ཐང་།  པེ་ཅིན། }ནས་སྔགས་:དྲན་པ་དང་\footnote{དྲན་པར་  སྣར་ཐང་། དྲན་པ་  པེ་ཅིན། }ལྡན་པར་ཕྱིན་ན་:ཆོམ་རྐུན་\footnote{ཆོམ་རྐུན་པ་  སྣར་ཐང་།  པེ་ཅིན། }དང་། སྟག་དང་། དོམ་དང་། དྲེད་དང་། སྦྲུལ་དང་། ནགས་ཀྱི་བ་\footnote{བར་  སྣར་ཐང་། }མེན་དང་། ཆུ་ལ་སྤྱོད་པ་རྣམས་ཀྱིས་བརྫི་བར་མི་ནུས་ཏེ། གང་གི་ཚེ་འཇིགས་པ་དང་བྲལ་བ་ན་མདུད་པ་སླར་དགྲོལ་བར་བྱའོ། །དེ་ལྟར་:ཡུན་རིང་\footnote{ཡུན་རིངས་  སྣར་ཐང་།  པེ་ཅིན། }བླ་མ་བསྟེན་ལས་གང་ཐོབ་པའི། །སྒྲུབ་ཐབས་ཆོ་ག་འདི་ནི་ལེགས་བཀོད་པས། །སྒྲོལ་མ་ཞེས་གྲགས་སྲས་བཅས་བཞུགས་པའི་གནས། །གྲུ་འཛིན་དཔལ་དུ་འགྲོ་རྣམས་མྱུར་འགྲོ་ཤོག །འཕགས་མ་སྒྲོལ་མའི་སྒྲུབ་ཐབས་དཔལ་མར་མེ་མཛད་ཡེ་ཤེས་ཀྱིས་:མཛད་པ་རྫོགས་སོ།\footnote{མཛད།  སྣར་ཐང་།  པེ་ཅིན། }། །།རྒྱ་གར་གྱི་མཁན་པོ་བུདྡྷ་ཀཱ་ར་བརྨ་དང་། བོད་:ཀྱི་ལོ་ཙཱ་\footnote{ཀྱི་ལོཙྪ་  སྣར་ཐང་།  པེ་ཅིན། }བ་དགེ་སློང་ཆོས་ཀྱི་ཤེས་རབ་ཀྱིས་བསྒྱུར་ཅིང་ཞུས་ཏེ་གཏན་ལ་ཕབ་པའོ། །
